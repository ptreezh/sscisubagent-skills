# Paradigm六个组件详解

> **前置阅读**：建议先阅读 `INDEX.md` 了解基本概念。

## 1. 现象（Phenomenon）

### 定义
研究的核心事件、发生的事情、行动/互动所指向的中心议题。

### 识别方法

**方法1：频率分析**
- 寻找访谈中反复出现的主题
- 统计概念/范畴的出现频次

**方法2：连接度分析**
- 找到连接最多其他范畴的核心范畴
- 使用工具：`identify_categories.py` 查看 `core_categories` 字段

**方法3：参与者视角**
- 识别参与者最关注的问题
- 询问"这个研究最核心的问题是什么？"

### 案例：大学生学业适应研究

**现象**：学业适应困难

**定义**：新生在大学学习过程中遇到的学习方法、课程难度、时间管理等方面的挑战

**证据**：
- 访谈中80%的学生提到"不适应"、"困难"、"压力大"
- 45个相关文本片段
- 连接了其他4个主要范畴

**工具输出示例**：
```json
{
  "phenomenon": {
    "category": "学业适应困难",
    "frequency": 45,
    "connections": 12,
    "proportion": 0.19
  }
}
```

## 2. 因果条件（Causal Conditions）

### 定义
导致现象发生的事件、情境或背景因素。回答"为什么会发生？"

### 识别方法

**方法1：关键词识别**
```python
causal_keywords = ['导致', '引起', '产生', '造成', '带来', '促使', 
                   '使得', '因为...所以', '由于...因此']
```

**方法2：时间序列分析**
- A在时间上先于B
- A的出现与B的出现有稳定关联

**方法3：逻辑推理**
- 如果A不存在，B是否还会发生？
- A的强度变化是否影响B的强度？

### 案例

**因果条件1：中学与大学学习方式差异**
```
中学：被动接受、教师主导、题海战术
  ↓ 导致
大学：主动学习、自主管理、理解为主的不适应
  ↓ 导致
学业适应困难
```

**数据证据**：
> "高中时老师会把所有知识点都讲得很细，我们只要记住就行。但大学老师讲得很快，很多东西要自己理解。我一开始完全不适应，成绩也不好。"（学生A）

**关系强度**：0.85（高）
- 9/10受访者提到
- 时间序列清晰
- 逻辑合理

**因果条件2：课程难度突然提升**
- 专业课抽象程度高
- 知识量大、进度快
- 缺乏基础知识衔接

**关系强度**：0.78（中高）

## 3. 语境（Context）

### 定义
现象发生的特定情境、背景条件。回答"在什么条件下？"、"在哪里？"

### 与因果条件的区别

| 维度 | 因果条件 | 语境 |
|-----|---------|------|
| 问题 | 为什么发生？ | 在哪里发生？ |
| 作用 | 直接导致现象 | 提供发生环境 |
| 案例 | 学习方法差异 | 重点大学环境 |

### 案例

**语境1：重点大学的学业竞争环境**
- 同学普遍学习能力强
- 课程要求高、淘汰率高
- 绩点竞争激烈

**语境2：远离家乡的独立生活**
- 缺少家长监督和支持
- 需要自己处理生活琐事
- 孤独感和思乡情绪

**语境3：大一新生的特殊阶段**
- 身份转换期（中学生→大学生）
- 社交网络重建期
- 职业规划迷茫期

## 4. 中介条件（Intervening Conditions）

### 定义
促进或限制行动策略的更广泛的结构性条件。

### 识别方法

**方法1：寻找转折词**
- "但是"、"然而"、"虽然"

**方法2：对比分析**
- 为什么同样的行动在不同情况下效果不同？
- 成功案例 vs 失败案例的差异

**方法3：结构性因素分析**
- 资源、权力、文化等因素

### 案例

**中介条件1：个人性格特质**
```
寻求帮助行为
  ↓ 受到影响
个人性格
  - 外向者：更主动、更频繁
  - 内向者：较少求助、倾向自学
```

**数据证据**：
> "我性格比较外向，遇到问题就会去找老师或同学问。我觉得这没什么不好意思的。"（学生E，外向）

> "我比较内向，不太好意思去问老师。我更喜欢自己查资料解决问题。"（学生F，内向）

**中介条件2：社会支持网络**
- 有良好支持：策略更有效，适应更快
- 缺乏支持：策略效果有限，适应缓慢

**中介条件3：学校资源可及性**
- 学业辅导中心的开放时间
- 教师答疑的便利程度
- 心理咨询服务的可获得性

## 5. 行动/互动策略（Action/Interaction Strategies）

### 定义
人们为应对现象而采取的有目的、有意识的行动。

### 识别方法

**方法1：动词短语识别**
- "寻求"、"调整"、"建立"、"应对"

**方法2：询问**
- "人们做了什么来应对？"
- "采取了哪些行动？"

**方法3：成功/失败分析**
- 哪些策略有效？
- 哪些策略无效？

### 案例

**策略1：主动寻求学业支持**
- 向教师请教问题
- 参加学习小组
- 寻找学长学姐指导

**策略2：调整学习方法**
- 从题海战术转向理解为主
- 制定学习计划
- 使用时间管理工具

**策略3：建立社会支持网络**
- 结交志同道合的朋友
- 加入学习型社团
- 保持与家人的沟通

**策略4：心态调整**
- 接受"不完美"的自己
- 降低不合理期望
- 培养积极归因方式

**工具支持**：
```bash
python scripts/construct_paradigm.py --input relationships.json
# 查看 details.paradigm.action_strategies 字段
```

## 6. 结果（Consequences）

### 定义
行动/互动策略产生的结果，可以是预期的或非预期的。

### 识别方法

**方法1：关键词识别**
- "结果"、"导致"、"最终"

**方法2：时间序列分析**
- 什么在行动之后发生？

**方法3：结果分类**
- 短期 vs 长期
- 预期 vs 非预期
- 积极 vs 消极

### 案例

**积极结果**：
- 学业成绩提升（预期、短期）
- 学习自信心增强（预期、中期）
- 建立了有效的学习方法（预期、长期）
- 形成了良好的社交网络（非预期、积极）

**消极结果**：
- 部分学生过度依赖他人帮助（非预期、消极）
- 时间管理仍然困难（部分未达成）
- 焦虑情绪未完全缓解（部分未达成）

**意外结果**：
- 发现了新的兴趣方向（非预期、积极）
- 建立了深厚的友谊（非预期、积极）
- 学会了自我反思和成长（非预期、积极）

## Paradigm构建流程

### 第一步：识别核心现象
```bash
python scripts/identify_categories.py --input codes.json --output categories.json
```
- 选择最能概括研究主题的范畴
- 确保现象有充分的数据支持

### 第二步：识别因果条件和语境
```bash
python scripts/build_relationships.py --input categories.json --output relationships.json
```
- 区分"为什么"（因果）和"在哪里"（语境）
- 寻找多重因果条件

### 第三步：识别行动策略
- 列出所有应对策略
- 分析策略的有效性

### 第四步：识别中介条件
- 分析为什么同样的策略效果不同
- 识别结构性限制因素

### 第五步：识别结果
- 区分短期和长期结果
- 识别预期和非预期结果

### 第六步：整合Paradigm模型
```bash
python scripts/construct_paradigm.py --input relationships.json --output paradigm.json
```
- 检查逻辑链条的完整性
- 验证每个组件的数据支持

## 验证标准

### Strauss & Corbin (1998) 的标准

**必要性**：A是B发生的必要条件吗？

**充分性**：A是B发生的充分条件吗？

**时间先后**：A在时间上是否先于B？

**证据充分性**：有多少数据支持？

**替代解释**：是否有其他因素也能解释B？
