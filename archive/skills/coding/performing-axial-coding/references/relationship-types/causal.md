# 因果关系详解

> **前置阅读**：建议先阅读 `INDEX.md` 了解关系类型概览。

## 定义

因果关系描述一个范畴直接导致另一个范畴的出现或变化。回答"为什么会发生？"

**Paradigm组件**：因果条件 → 现象

---

## 识别方法

### 方法1：关键词识别
```python
causal_keywords = ['导致', '引起', '产生', '造成', '带来', '促使',
                   '使得', '令', '让', '因为...所以', '由于...因此']
```

### 方法2：时间序列分析
- A在时间上先于B
- A的出现与B的出现有稳定关联

### 方法3：逻辑推理
- 如果A不存在，B是否还会发生？
- A的强度变化是否影响B的强度？

## 典型案例

### 案例1：学习方法差异 → 学业适应困难

**因果链条**：
```
中学学习方式（被动接受、题海战术）
    ↓ 导致
大学学习方式（主动理解、应用为主）的不适应
    ↓ 导致
学业适应困难（成绩下降、焦虑）
```

**数据证据**：
> "高中时老师会把所有知识点都讲得很细，我们只要记住就行。但大学老师讲得很快，很多东西要自己理解。我一开始完全不适应，成绩也不好。"（学生A）

**关系强度**：0.85（高）
- 9/10受访者提到
- 时间序列清晰
- 逻辑合理

**工具输出**：
```json
{
  "relation_type": "causal",
  "source": "学习方法差异",
  "target": "学业适应困难",
  "strength": 0.85,
  "evidence_count": 9,
  "confidence": "high"
}
```

### 案例2：课程难度提升 → 学业压力

**数据证据**：
> "大一的高数和物理太难了，比高中难太多。我每天都很焦虑，担心考试不及格。"（学生C）

**关系强度**：0.78（中高）

## 验证标准

### Strauss & Corbin (1998) 的标准

**必要性**：A是B发生的必要条件吗？
- 如果A不存在，B是否还会发生？

**充分性**：A是B发生的充分条件吗？
- A的存在是否总是导致B？

**时间先后**：A在时间上是否先于B？

**证据充分性**：有多少数据支持？（至少3个片段）

**替代解释**：是否有其他因素也能解释B？

## 常见错误

### 错误1：混淆因果和相关

❌ **错误**："成绩好的学生往往自信强，所以成绩好导致自信强。"

**问题**：忽略了反向因果（自信强也可能导致成绩好）

✅ **正确**：
- 分析时间序列
- 考虑双向因果
- 识别互动关系（见 `interaction.md`）

### 错误2：忽略中介变量

❌ **错误**："学习方法差异直接导致成绩下降。"

**问题**：忽略了中间环节

✅ **正确**：
```
学习方法差异 → 学习效率低 → 知识掌握不足 → 成绩下降
```
