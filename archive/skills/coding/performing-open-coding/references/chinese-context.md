# 中文质性研究的特殊性

> **渐进式披露**：本文档聚焦中文研究的文化和语言特点。理论基础见 `theory.md`，实践案例见 `examples.md`。

## 文化维度

### 关系导向（Guanxi）

**特征**：
- 中国社会重视人际关系网络
- "关系"不仅是连接，更是资源和义务

**编码建议**：
```
重点关注：
- "建立关系"、"维护关系"、"利用关系"
- "人情往来"、"面子互动"
- "圈子"、"网络"、"人脉"

示例概念：
- `建立师生关系` - 与教师建立超越课堂的关系
- `维护同学关系` - 通过互惠维持同学情谊
- `动用人脉资源` - 利用关系网络获取资源
```

### 集体主义

**特征**：
- 强调群体和谐
- 个人服从集体

**编码建议**：
```
重点关注：
- "融入集体"、"适应群体"
- "集体决策"、"群体压力"
- "从众行为"、"维护和谐"

示例概念：
- `融入宿舍集体` - 努力成为宿舍群体的一员
- `适应班级规范` - 遵循班级的非正式规则
- `避免特立独行` - 不做与众不同的行为
```

### 面子文化

**特征**：
- 重视社会评价和声誉
- 避免丢面子的情境

**编码建议**：
```
重点关注：
- "维护面子"、"给面子"、"丢面子"
- "避免尴尬"、"保全颜面"
- "顾及他人感受"

示例概念：
- `避免当众出错` - 不在公开场合暴露不足
- `维护教师权威` - 不质疑老师以保全面子
- `委婉表达异议` - 间接方式表达不同意见
```

### 权威尊重

**特征**：
- 尊重权威和长辈
- 层级关系明显

**编码建议**：
```
重点关注：
- "尊重权威"、"服从安排"
- "等级意识"、"上下关系"
- "礼貌规范"

示例概念：
- `尊重教师权威` - 认可教师的专业地位
- `遵循长辈建议` - 接受年长者的指导
- `保持礼貌距离` - 维持师生间的适当距离
```

## 语言维度

### 词汇特点

#### 双音节结构

**特征**：中文多为双音节词

**编码建议**：
```
✅ 优先使用双音节：
- "学习"（而非"学"）
- "帮助"（而非"帮"）
- "关系"（而非"系"）

✅ 四字结构更佳：
- "寻求帮助"
- "建立关系"
- "适应环境"
```

#### 动词丰富性

**特征**：中文动词表达细腻

**常用行动动词**：
```
寻求、建立、适应、应对、处理、解决、
获得、提供、学习、制定、执行、调整、
保持、克服、管理、理解、融入、维护
```

**编码示例**：
```
✅ "寻求教师指导"（主动获取）
✅ "获得教师支持"（被动接受）
✅ "建立同伴关系"（创建新关系）
✅ "维护同学关系"（保持已有关系）
```

#### 隐含意义

**特征**：中文常有隐含意义

**示例**：
- "不好意思问老师" → 概念：`避免寻求帮助`（隐含：面子顾虑）
- "大家都这样做" → 概念：`从众行为`（隐含：集体压力）

**编码建议**：
- 识别表面行为
- 理解深层动机
- 两者都编码

#### 停用词定制

**中文研究专用停用词**：
```python
CHINESE_RESEARCH_STOPWORDS = {
    # 基础停用词
    '的', '了', '在', '是', '我', '有', '和',
    
    # 研究场景停用词
    '访谈', '受访者', '研究', '调查',
    
    # 口语化停用词
    '就是', '然后', '那个', '这个', '嗯', '啊'
}
```

### 相似度计算优化

#### 中文语义相似度

**方法选择**：
- **TF-IDF + 余弦相似度**：当前使用，快速有效
- **Word2Vec**：更准确，但需要预训练模型
- **BERT**：最准确，但计算成本高

**推荐**：
- 小规模数据（<1000个概念）：TF-IDF
- 大规模数据：考虑Word2Vec

### 实践1：迭代精炼

**第一轮**：快速编码

**第二轮**：人工精炼


**第四轮**：质量验证
```bash
python scripts/validate_codes.py --input v3.json --output validation.json
```

### 实践2：团队协作

**编码一致性**：
- 使用统一的编码词典
- 定期团队讨论
- 交叉验证编码

**工具辅助**：
```bash
# 合并多人编码结果
cat person1.json person2.json | python scripts/compare_codes.py --input - --output merged.json
```

### 实践3：理论敏感性

**保持开放**：
- 不预设理论框架
- 让概念从数据中涌现
- 使用备忘录记录理论洞见

**文化敏感**：
- 理解中文表达的隐含意义
- 识别文化特有的现象
- 避免西方理论的生搬硬套
