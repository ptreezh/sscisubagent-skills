# 学术引用格式化技能

## 🎯 功能概述

提供多种学术引用格式的自动格式化服务，特别优化中文社科研究的GB/T 7714标准。

## 📚 支持的引用格式

### 中文格式
- **GB/T 7714-2015**: 中国国家标准文献著录规则
- **中国社会科学**: 中文学术期刊常用格式
- **学报格式**: 大学学报标准格式

### 英文格式
- **APA 7th**: 美国心理学会格式
- **MLA 9th**: 现代语言协会格式
- **Chicago 17th**: 芝加哥格式
- **Harvard**: 哈佛格式
- **IEEE**: 工程技术格式

## 🔧 核心功能

### 1. 自动格式识别
```python
# 输入示例
张三. (2020). 在线学习对学生学习效果的影响研究. 教育研究, 41(5), 23-35.

# 自动识别为:
- 格式: 中文学术期刊
- 标准类型: 类似GB/T 7714
- 文献类型: 期刊文章
```

### 2. 格式转换服务
```bash
# GB/T 7714 → APA
张三. (2020). 在线学习对学生学习效果的影响研究. 教育研究, 41(5), 23-35.
↓
Zhang, S. (2020). The impact of online learning on student learning outcomes. Educational Research, 41(5), 23-35.
```

### 3. 格式验证和修复
```python
# 检查常见错误
- 作者姓名格式
- 期刊名称缩写
- 年份位置
- 卷期页码格式
- DOI格式
- URL格式
```

## 📋 文献类型支持

### 期刊文章
```bash
# GB/T 7714格式
[序号] 作者. 题名[J]. 期刊名, 年, 卷(期): 起止页码.

# 示例
[1] 李华, 王芳. 基于深度学习的图像识别技术综述[J]. 计算机学报, 2020, 43(6): 1121-1135.
```

### 图书
```bash
# GB/T 7714格式
[序号] 作者. 书名[M]. 版本. 出版地: 出版者, 出版年: 引文页码.

# 示例
[2] 刘明. 数据结构与算法[M]. 2版. 北京: 清华大学出版社, 2019: 45-67.
```

### 学位论文
```bash
# GB/T 7714格式
[序号] 作者. 题名[D]. 保存地: 保存单位, 年份.

# 示例
[3] 陈晓. 大数据时代个人信息保护法律制度研究[D]. 北京: 北京大学, 2021.
```

### 会议论文
```bash
# GB/T 7714格式
[序号] 作者. 题名[C]//编者. 文集名. 出版地: 出版者, 出版年: 起止页码.

# 示例
[4] 赵强. 人工智能在教育中的应用研究[C]//中国教育学会. 教育信息化国际会议论文集. 上海: 华东师范大学出版社, 2020: 156-162.
```

### 网络资源
```bash
# GB/T 7714格式
[序号] 作者. 题名[EB/OL]. (更新日期)[引用日期]. URL.

# 示例
[5] 教育部. 教育信息化2.0行动计划[EB/OL]. (2018-04-18)[2023-10-15]. http://www.moe.gov.cn/srcsite/A16/s7062/201804/t20180425_334188.html.
```

## 🏫 中文学术特殊处理

### 1. 中文姓名处理
```bash
# 单姓单名
张三 → Zhang, S.

# 单姓双名
张小明 → Zhang, X. M.

# 复姓
欧阳修 → Ouyang, X.

# 英文姓名中文翻译
John Smith → 约翰·史密斯
```

### 2. 期刊名称标准化
```python
# 期刊全称 → 标准缩写
"教育研究" → "Educ. Res."
"计算机学报" → "J. Comput. Sci. Technol."
"心理学报" → "Acta Psychol. Sin."
```

### 3. 中英文混合引用
```bash
# 同一文献的中英文版本
[1] 张三. 在线学习研究[J]. 教育研究, 2020, 41(5): 23-35.
Zhang S. Research on online learning[J]. Educational Research, 2020, 41(5): 23-35.

# 处理原则
- 优先使用原文语言格式
- 保持一致性
- 标注中英文版本关系
```

## 📊 格式检查清单

### GB/T 7714格式检查
- [ ] 作者姓名格式正确
- [ ] 文献类型标识符正确[J][M][D][C][EB/OL]
- [ ] 期刊名称完整或标准缩写
- [ ] 年份位置正确
- [ ] 卷期格式正确
- [ ] 页码范围格式正确
- [ ] DOI格式正确(如果有)
- [ ] URL格式正确(网络资源)
- [ ] 标点符号使用正确

### 常见错误修正
```python
# 错误示例及修正
❌ 张三,在线学习研究,教育研究,2020,41(5):23-35.
✅ 张三. 在线学习研究[J]. 教育研究, 2020, 41(5): 23-35.

❌ Wang, J., 2020. "Online Learning." Journal of Education.
✅ Wang, J. (2020). Online learning. Journal of Education, 100(1), 1-15.
```
