# 中文学术论文结构技能

## 🎯 核心目标（最高优先级）
帮助中文社会科学研究者构建符合学术规范、逻辑清晰、表达准确的学术论文结构。

## 📋 必须首先掌握的结构规范

### 1. 中文学术论文标准结构
**必须遵循**：
```
题目 → 作者 → 摘要 → 关键词 → 引言 → 文献综述 → 方法 → 结果 → 讨论 → 结论 → 参考文献
```

### 2. IMRAD结构本土化适配
**中文特色**：
- **引言**：突出研究意义和实践价值
- **方法**：详细说明本土化适配
- **结果**：结合中国文化背景解释
- **讨论**：强调理论贡献和实践意义

### 3. 质量控制的黄金标准
**必须检查**：
- 逻辑连贯性（论点→论据→结论）
- 学术规范性（引用、格式、伦理）
- 表达准确性（专业术语、语法）
- 实用价值（理论贡献、实践意义）

## 🛠️ 核心写作技能

### 1. 引言写作（最重要）
**核心要素**：
```python
def structure_introduction(research_topic):
    """引言结构模板"""
    return {
        'opening_hook': f"当前{research_topic}领域面临的重要挑战",
        'research_gap': "现有研究的不足和空白",
        'research_question': "明确的研究问题",
        'research_significance': "理论意义和实践价值",
        'research_objectives': "具体研究目标",
        'paper_structure': "论文结构安排"
    }

# 中文特色表达模板
chinese_templates = {
    'significance': [
        "具有重要的理论价值和现实意义",
        "为解决...问题提供了新的思路",
        "对推动...领域发展具有重要意义"
    ],
    'gap': [
        "现有研究存在以下不足",
        "鲜有学者关注...问题",
        "在...方面的研究相对薄弱"
    ]
}
```

## 📊 质量检查清单

### 结构完整性检查
- [ ] 题目是否准确反映研究内容
- [ ] 摘要是否包含研究目的、方法、结果、结论
- [ ] 关键词是否准确全面
- [ ] 章节结构是否完整合理
- [ ] 逻辑关系是否清晰连贯

### 内容质量检查
- [ ] 研究问题是否明确
- [ ] 文献综述是否全面
- [ ] 研究方法是否适切
- [ ] 数据分析是否严谨
- [ ] 结论是否基于结果

### 学术规范检查
- [ ] 引用格式是否符合GB/T 7714
- [ ] 学术语言是否规范
- [ ] 图表格式是否标准
- [ ] 研究伦理是否遵守
- [ ] 抄袭检查是否通过
