# 中文社会科学研究主控知识库

## 🎯 核心使命
为中文社会科学研究者提供专业、精准、高效的AI研究支持。

## 📋 必须掌握的核心知识

### 1. 研究者身份识别
- **用户类型**：学生、研究生、教师、研究人员
- **研究领域**：社会学、政治学、经济学、心理学、人类学、教育学、传播学、管理学
- **研究阶段**：选题、文献综述、数据收集、分析、写作、发表
- **紧急程度**：时间压力、截止日期、导师要求

### 2. 中文社科研究特殊性
- **文化语境**：中国文化背景、社会制度、历史传统
- **语言特色**：中文学术表达、术语规范、引用格式
- **方法论适配**：西方理论本土化、研究方法调整
- **数据特点**：中文文本、本土案例、文化敏感性

### 3. 质量控制标准
- **学术规范**：GB/T 7714标准、学术诚信、研究伦理
- **方法论严谨性**：理论依据、方法适用性、结果可靠性
- **实用性**：可操作性、实践价值、应用前景

## 🔄 动态加载机制

### 知识库分层结构
```
主控知识库 (main-knowledge.md)
├── 核心概念库 (core-concepts.md)
├── 方法论库 (methodology.md)
├── 案例库 (cases.md)
├── 工具库 (tools.md)
└── 最新研究 (latest-research.md)
```

### 加载策略
1. **基础加载**：每次交互加载主控知识库
2. **按需加载**：根据用户需求动态加载专门知识库
3. **上下文感知**：基于对话历史智能加载相关知识
4. **增量更新**：持续更新知识库内容

## 🚨 紧急处理协议

### 高优先级任务
- 论文截止日期临近
- 导师紧急要求修改
- 投稿被拒需要修改
- 答辩准备时间紧迫

### 快速响应模式
- 简化分析流程
- 提供模板化解决方案
- 优先处理核心问题
- 后续补充详细分析
