# 理论框架整合指南

> **渐进式学习**：本文档提供理论整合的概览和方法。

## 什么是理论整合

理论整合（Theory Integration）是将所有范畴、关系和发现整合为系统性的理论框架，形成可以发表的理论贡献。

**目标**：
- 提炼理论命题
- 构建概念框架
- 解释作用机制
- 界定理论边界

---

## 四个关键任务

### 1. 提炼理论命题

**定义**：基于范畴关系形成的可检验的理论陈述

**方法**：
```bash
python scripts/integrate_theory.py \
  --categories categories.json \
  --relations relationships.json
# 查看 details.propositions
```

**命题类型**：
- 因果命题："A会导致B"
- 条件命题："在X条件下，A会影响B"
- 策略命题："通过A可以达成B"

**案例**：
```
命题1：学习方法差异会导致学业适应困难的发生。
命题2：在个人性格外向的条件下，学生更倾向于主动寻求学业支持。
命题3：通过调整学习策略可以提升学业成绩和学习自信心。
```

详见：`propositions.md`

---

### 2. 构建概念框架

**定义**：展示核心概念及其关系的系统性图示

**结构**：
```
前因（Antecedents）
    ↓
核心概念（Core Concept）
    ↓
作用机制（Mechanisms）
    ↓
结果（Outcomes）
```

**案例**：
```
前因：学习方法差异、课程难度提升
  ↓
核心概念：学业适应困难
  ↓
机制：主动寻求支持、调整学习策略
  ↓
结果：成绩提升、自信增强
```

---

### 3. 解释作用机制

**定义**：深入分析现象背后的"为什么"和"如何"

**要点**：
- 不仅描述"发生了什么"
- 更要解释"为什么发生"
- 以及"如何发生"

**案例**：
```
机制：社会支持的缓冲作用

解释：社会支持通过以下机制缓解适应困难：
1. 信息支持：提供学习方法和经验
2. 情感支持：减轻焦虑和压力
3. 工具支持：提供学习资源和帮助
4. 评价支持：增强自信和动力

这一机制在中国文化中尤为重要，因为关系网络
是个体获取资源的主要途径。
```

---

### 4. 界定理论边界

**定义**：明确理论的适用范围和限制

**要点**：
- 适用的人群和情境
- 不适用的情况
- 理论的局限性

**案例**：
```
适用范围：
- 重点大学的新生
- 学业适应困境
- 中国文化背景

不适用情况：
- 非学业适应问题（如人际关系）
- 高年级学生（已度过适应期）
- 西方文化背景（关系网络作用不同）

局限性：
- 样本集中在某一所大学
- 访谈时间跨度有限
- 未考虑专业差异
```

## 使用工具整合

```bash
python scripts/integrate_theory.py \
  --categories categories.json \
  --relations relationships.json \
  --data data.json \
  --output theory.json
```

**输出**：
```json
{
  "summary": {
    "core_category": "学业适应困难",
    "propositions_count": 8,
    "framework_completeness": 0.85
  },
  "details": {
    "propositions": [...],
    "conceptual_framework": {...},
    "mechanisms": {...},
    "boundaries": {...}
  }
}
```

## 常见问题

**问题1**：理论命题过于宽泛
- 解决：增加条件限定，提高精确性

**问题2**：概念框架过于复杂
- 解决：简化次要概念，突出核心

**问题3**：机制解释不够深入
- 解决：追问"为什么"和"如何"
