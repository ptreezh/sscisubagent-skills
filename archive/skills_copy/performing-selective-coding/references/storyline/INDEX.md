# 故事线构建指南

> **渐进式学习**：本文档提供故事线构建的概览和方法。

## 什么是故事线

故事线（Storyline）是将理论以叙事形式呈现的连贯描述，展示核心现象如何发展、参与者如何行动、结果如何产生。

**Strauss & Corbin (1998) 的定义**：
> "故事线是理论的叙事表达，它将抽象的范畴和关系转化为可理解的故事。"

---

## 故事线的四个要素

### 1. 时间线（Timeline）
- 现象的发展阶段
- 关键事件和转折点
- 时间顺序和因果顺序

### 2. 行动者（Actors）
- 主要参与者
- 角色和作用
- 互动模式

### 3. 因果链条（Causal Chain）
- 核心因果关系
- 条件和机制
- 结果和影响

### 4. 叙事结构（Narrative Structure）
- 开端：现象的出现
- 发展：应对和互动
- 结果：最终状态

## 构建方法

### 使用工具辅助

```bash
python scripts/construct_storyline.py \
  --input data.json \
  --categories categories.json \
  --relations relationships.json \
  --core-category "学业适应困难"
```

**工具输出**：
```json
{
  "summary": {
    "core_category": "学业适应困难",
    "key_events_count": 8,
    "actors_count": 3
  },
  "details": {
    "storyline_text": "本研究聚焦于'学业适应困难'这一核心现象..."
  }
}
```

### 人工精炼

**第一步：确定叙事视角**
- 参与者视角（第一人称）
- 研究者视角（第三人称）
- 理论视角（抽象概念）

**第二步：构建叙事框架**
```
开端 → 发展 → 高潮 → 结局
```

**第三步：填充细节**
- 添加具体案例和引语
- 丰富因果关系的解释
- 增强叙事的生动性

## 典型故事线模板

```
大学新生由于[因果条件]，经历了[核心现象]。

在[语境条件]的背景下，受到[中介条件]的影响，
学生采取了[行动策略]来应对这一困境。

通过[具体行动]，学生逐渐[发展过程]，
最终[结果]。

这一过程揭示了[理论洞察]。
```

## 案例预览

**研究**：大学生学业适应

**故事线**（简化版）：
```
大学新生由于学习方法差异和课程难度提升，经历了学业适应困难。

在重点大学的竞争环境下，受到个人性格和社会支持网络的影响，
学生采取了主动寻求学业支持、调整学习策略等行动。

通过建立师生关系、参加学习小组、改进学习方法，
学生逐渐克服了适应困难，不仅成绩提升，
还建立了有效的学习方法和社交网络。

这一过程揭示了主动适应的重要性和社会支持的关键作用。
```

## 质量标准

**逻辑清晰**：
- [ ] 因果关系明确
- [ ] 时间顺序合理
- [ ] 没有逻辑跳跃

**内容完整**：
- [ ] 包含所有核心范畴
- [ ] 涵盖关键事件
- [ ] 展示主要关系

**叙事连贯**：
- [ ] 故事流畅自然
- [ ] 过渡自然
- [ ] 易于理解

**理论深度**：
- [ ] 不仅是描述，更有解释
- [ ] 揭示理论洞察
- [ ] 有学术价值

## 常见问题

**问题1**：故事线过于平铺直叙
- 解决：增加转折点和冲突

**问题2**：故事线与数据脱节
- 解决：添加具体引语和案例

**问题3**：故事线缺乏理论深度
- 解决：增加理论解释和洞察
