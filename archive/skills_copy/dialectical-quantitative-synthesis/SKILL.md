# 辩证定量综合分析技能

## 核心功能

### 1. 理论指导数据收集
- **理论框架指导**：用马克思主义理论指导数据收集方向和重点
- **概念操作化**：将马克思主义理论概念操作化为可测量指标
- **变量关系假设**：基于理论提出变量间关系的假设
- **数据质量标准**：建立基于理论要求的数据质量标准

### 2. 定量验证理论假设
- **假设检验**：用定量数据验证马克思主义理论假设
- **因果关系分析**：运用计量经济学方法分析因果关系
- **统计显著性检验**：验证理论关系的统计显著性
- **效应量计算**：计算理论变量的实际效应大小

### 3. 定性洞察指导解释
- **理论解释框架**：用马克思主义理论解释定量结果
- **机制分析**：揭示数据背后的社会机制和规律
- **异常值处理**：用理论指导异常值的处理和解释
- **结果外推**：基于理论判断定量结果的外推合理性

### 4. 综合分析报告
- **理论数据融合**：形成理论与数据的综合分析结果
- **多层次解释**：提供多层次、多维度的分析解释
- **实践指导**：基于综合分析提供实践指导建议
- **理论发展**：促进马克思主义理论的创新发展
