# 行动者识别参考文档 (Actor Identification Reference)

## 概述

行动者识别是ANT分析的第一步，需要识别人类和非人类行动者，并分析其特征和关系。

## 人类行动者识别

### 个体行动者
- **定义**：个人作为网络节点
- **识别方法**：
  - 姓名识别（中文姓名、英文姓名）
  - 角色识别（职位、职能、身份）
  - 行为识别（行动、决策、影响）
- **特征分析**：
  - 行动能力
  - 资源控制
  - 网络位置
  - 影响力评估

### 组织行动者
- **定义**：机构、企业、团体等作为网络节点
- **识别方法**：
  - 机构名称
  - 职能定位
  - 结构特征
- **特征分析**：
  - 规模大小
  - 资源掌控
  - 网络中心性
  - 行动能力

### 群体行动者
- **定义**：社区、网络、联盟等群体性实体
- **识别方法**：
  - 群体标识
  - 成员构成
  - 共同目标
- **特征分析**：
  - 群体凝聚力
  - 代表机制
  - 行动一致性

## 非人类行动者识别

### 技术行动者
- **定义**：设备、系统、平台等技术实体
- **识别方法**：
  - 功能分析
  - 依赖关系
  - 作用机制
- **特征分析**：
  - 行动能力
  - 约束能力
  - 转译能力

### 制度行动者
- **定义**：政策、法规、规范等制度性实体
- **识别方法**：
  - 规则识别
  - 约束分析
  - 权威来源
- **特征分析**：
  - 约束力
  - 合法性
  - 影响范围

### 物质行动者
- **定义**：资金、资源、设施等物质实体
- **识别方法**：
  - 资源类型
  - 控制关系
  - 流动路径
- **特征分析**：
  - 稀缺性
  - 可控性
  - 流动性

## 识别策略

### 文本分析策略
```python
def identify_actors_from_text(text):
    """
    从文本中识别行动者
    """
    human_actors = extract_human_actors(text)
    nonhuman_actors = extract_nonhuman_actors(text)
    return {
        'human_actors': human_actors,
        'nonhuman_actors': nonhuman_actors,
        'relationships': identify_relationships(human_actors + nonhuman_actors)
    }
```

### 关系网络策略
- 通过关系识别行动者
- 分析行动者间的关系强度
- 评估关系的稳定性

### 参与观察策略
- 现场观察行动者互动
- 记录行动者关系建立过程
- 评估行动者影响力

## 常见问题

### 人类行动者识别问题
- **问题**：难以区分个体和组织
- **解决**：明确行动主体的独立性

### 非人类行动者识别问题
- **问题**：过度拟人化非人类实体
- **解决**：关注非人类实体的行动能力而非意图

### 关系分析问题
- **问题**：关系强度评估主观
- **解决**：使用多维度指标评估
