# fsQCA校准指导

## 任务目标
将原始数据转化为理论上有意义的模糊集隶属度，确保校准结果符合理论预期并尊重数据特征。

## 核心任务
1. **数据特征分析** - 连续/离散/分类变量评估
2. **校准方法选择** - 直接/间接/阈值/插值等
3. **参数设定** - 锚点、阈值、隶属度函数
4. **质量检验** - 理论一致性、数据拟合度

## 简化校准流程
1. **数据评估** → 变量类型、分布特征、特殊值
2. **方法选择** → 基于数据特征和理论需求
3. **参数设定** → 具体校准参数
4. **质量检验** → 一致性、拟合度、可行性

## 主要校准方法
- **直接校准**: 3个理论锚点（完全隶属、交叉点、完全不隶属）
- **阈值校准**: 上下阈值定义隶属度
- **插值校准**: 多锚点插值函数
- **间接校准**: 分类到隶属度映射

## 输出要求
- 每个变量的校准方法和参数
- 校准的理论依据
- 验证计划

## 质量标准
- 理论一致性：校准结果符合理论预期
- 数据拟合度：充分利用数据信息
- 操作可行性：可重复且稳定

## 关键术语
- **隶属度**: 元素隶属于集合的程度（0到1）
- **锚点**: 校准中的关键参考点
- **t-范数**: 模糊集中的AND操作
