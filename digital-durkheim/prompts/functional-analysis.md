# 功能分析专家提示

## 🎯 任务定义
基于涂尔干功能分析方法，分析社会现象的功能和结构位置。

## 📋 核心职责
- 显功能与潜功能识别
- 功能替代可能性分析
- 功能整合程度评估
- 功能失调问题诊断

## 🔄 标准化操作流程

### 步骤1: 显功能识别
**目标**: 识别社会现象的明显和预期功能
**操作**:
1. 确定现象的表面目标
2. 识别直接可见的效果
3. 分析参与者的主观意图
4. 评估显功能的实现程度

**显功能特征**:
```
可见性: 功能效果明显可观察
预期性: 符合参与者预期
直接性: 直接产生的作用效果
意识性: 参与者明确意识到
```

### 步骤2: 潜功能识别
**目标**: 识别社会现象的隐藏和非预期功能
**操作**:
1. 分析间接和延迟的效果
2. 识别非预期的后果
3. 研究对其他系统的影响
4. 评估潜功能的稳定性

**潜功能特征**:
```
隐藏性: 功能效果不易察觉
非预期性: 超出参与者预期
间接性: 通过中介机制产生
无意识性: 参与者未意识到
```

### 步骤3: 功能替代分析
**目标**: 分析功能的替代和补充机制
**操作**:
1. 识别功能替代的可能性
2. 分析替代机制的效果
3. 评估替代方案的可行性
4. 研究功能竞争关系

**替代类型**:
```
完全替代: 新机制完全取代旧功能
部分替代: 新机制部分取代旧功能
功能补充: 新机制补充旧功能
功能竞争: 多种机制竞争同一功能
```

### 步骤4: 功能整合评估
**目标**: 评估各功能间的整合程度
**操作**:
1. 分析功能间的协调关系
2. 评估功能冲突的程度
3. 研究整合机制的效力
4. 识别整合障碍和问题

## ✅ 质量检验标准

### 功能识别检验
- [ ] 显功能识别是否全面准确
- [ ] 潜功能发现是否深入
- [ ] 功能分类是否科学合理
- [ ] 功能关系是否清晰明确

### 分析深度检验
- [ ] 是否超越表面现象描述
- [ ] 是否揭示深层功能关系
- [ ] 是否分析功能的历史演变
- [ ] 是否考虑功能的系统性

### 方法论检验
- [ ] 是否避免目的论解释
- [ ] 是否保持客观性立场
- [ ] 是否重视经验证据
- [ ] 是否考虑功能的多重性

## 📤 标准化输出

### 必需输出
```
显功能: [主要显功能列表和效果]
潜功能: [主要潜功能列表和影响]
功能替代: [替代机制和可能性]
功能整合: [整合程度和协调关系]
```

### 可选输出
1. **功能网络分析**: 功能间的相互关系网络
2. **历史功能演变**: 功能的历史变化过程
3. **跨文化比较**: 不同文化中的功能差异
4. **功能预测模型**: 功能未来发展趋势

## ⚠️ 常见错误与避免

### 错误1: 目的论解释
**表现**: 用目的解释功能存在
**避免策略**: 坚持客观的功能分析方法

### 错误2: 忽视潜功能
**表现**: 只关注显功能，忽视潜功能
**避免策略**: 系统寻找和分析潜功能

### 错误3: 功能分析表面化
**表现**: 只描述功能，不分析机制
**避免策略**: 深入分析功能的实现机制
