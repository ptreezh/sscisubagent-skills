# 数据计划详纲

## 1. 数据类型定义

### 1.1 定量数据
- **连续变量**: 年龄、收入、分数等
  - 特征: 无限多个可能值
  - 测量: 间隔或比率尺度
  - 分析: 参数统计方法
- **分类变量**: 性别、教育水平、分组等
  - 名义变量: 无序分类（如性别）
  - 有序变量: 有序分类（如教育水平）
  - 分析: 非参数统计方法

### 1.2 定性数据
- **文本数据**: 访谈记录、观察笔记、文档
  - 特征: 语言和文字信息
  - 分析: 主题分析、内容分析
- **视听数据**: 录音、录像、图像
  - 特征: 非文字信息
  - 分析: 现象学分析、视觉分析

### 1.3 混合数据
- **定量为主**: 定量数据为主，定性数据为辅
- **定性为主**: 定性数据为主，定量数据为辅
- **并重**: 定量与定性数据同等重要

## 2. 数据收集策略

### 2.1 主要数据收集
- **问卷调查**: 标准化、可量化
  - 优点: 大样本、标准化、易分析
  - 局限: 深度有限、可能存在社会期望偏差
  - 适用: 描述性、相关性研究
- **深度访谈**: 深入、详细
  - 优点: 深度信息、灵活性强
  - 局限: 样本量小、耗时、分析复杂
  - 适用: 探索性、解释性研究
- **观察**: 自然行为、环境
  - 优点: 真实行为、情境信息
  - 局限: 可能存在观察者偏差、伦理问题
  - 适用: 行为研究、情境研究
- **实验**: 控制条件、精确测量
  - 优点: 因果推断能力强
  - 局限: 生态效度、伦理限制
  - 适用: 实验性研究

### 2.2 次要数据收集
- **档案数据**: 官方记录、历史数据
  - 优点: 成本低、长期数据
  - 局限: 数据完整性、时效性
  - 适用: 历史研究、趋势分析
- **公开数据集**: 政府数据、研究数据
  - 优点: 可靠、标准化
  - 局限: 可能不符合研究需求
  - 适用: 验证性研究
- **文献数据**: 已发表研究、报告
  - 优点: 丰富的理论基础
  - 局限: 可能存在发表偏倚
  - 适用: 元底文献回顾

## 3. 数据收集工具

### 3.1 问卷设计
- **量表选择**: 已验证量表 vs 自制量表
  - 已验证量表: 信效度有保证
  - 自制量表: 针对性强但需验证
- **问题类型**: 
  - 封闭式: 选项固定，便于量化
  - 开放式: 自由回答，深度信息
  - 量表式: 李克特量表、语义差异量表
- **逻辑流程**: 问题顺序、跳转逻辑
  - 逻辑顺序: 从一般到具体
  - 跳转逻辑: 根据回答跳转到相关问题
- **预测试**: 小样本测试、修正
  - 适用: 发现问题、改进工具

### 3.2 访谈指南
- **开放性问题**: 鼓励详细回答
  - 特征: 开放、中性
  - 适用: 探索性研究
- **探针问题**: 深入挖掘信息
  - 特征: 引导、深入
  - 适用: 解释性研究
- **中性语言**: 選免引导性
  - 特征: 客观、中立
  - 适用: 所类型访谈
- **灵活性**: 适应不同受访者
  - 特征: 调整、适应
  - 适用: 所类型访谈

### 3.3 观察记录
- **结构化观察**: 预定义类别和指标
  - 特征: 标准化、可量化
  - 适用: 行为研究
- **非结构化观察**: 自由记录观察内容
  - 特征: 灵活、深入
  - 适用: 探索性研究
- **参与式观察**: 研究者参与活动
  - 特征: 深入、真实
  - 适用: 文化研究
- **非参与式观察**: 研究者外部观察
  - 特征: 客观、中立
  - 适用: 行为研究

## 4. 抽样与招募

### 4.1 抽样策略
- **目标总体**: 研究所关注的全部个体
- **可及总体**: 实际可接触的个体
- **样本**: 从总体中选择的研究对象
- **抽样方法**: 概率或非概率抽样

### 4.2 招募策略
- **招募渠道**: 线上、线下、专业机构
- **激励措施**: 礼品、金钱、知识贡献
- **筛选标准**: 入选/排除标准
- **招募时间**: 招募周期和节奏

## 5. 数据质量控制

### 5.1 数据收集质量
- **培训收集员**: 标准化程序、质量意识
- **标准化程序**: 统一指导语、操作流程
- **监控机制**: 现场监督、数据审核
- **数据验证**: 逻辑检查、范围检查

### 5.2 数据完整性
- **缺失数据处理**: 预防策略、处理方法
- **数据备份**: 多重备份、安全存储
- **时间管理**: 收集进度、截止时间
- **伦理合规**: 隐私保护、知情同意

## 6. 数据管理计划

### 6.1 数据存储
- **安全存储**: 加密、访问控制
- **格式标准**: 统一格式、兼容性
- **备份策略**: 定期备份、多地存储
- **存档计划**: 长期保存、共享策略

### 6.2 数据处理
- **数据清理**: 异常值、重复数据
- **数据编码**: 分类编码、缺失值编码
- **数据转换**: 标准化、聚合
- **数据链接**: 多源数据整合

## 7. 伦理考虑

### 7.1 知情同意
- **信息充分**: 研究目的、程序、风险
- **自愿参与**: 无强迫、可退出
- **理解能力**: 适合受试者理解水平
- **文档记录**: 签署同意书

### 7.2 隐私保护
- **匿名化**: 去除个人标识
- **保密性**: 数据访问限制
- **数据安全**: 阌止泄露措施
- **数据销毁**: 研究结束后处理

## 8. 实施时间表

### 8.1 阶段规划
- **准备阶段**: 工具开发、伦理审批
- **招募阶段**: 参试者招募、筛选
- **收集阶段**: 数据收集、质量监控
- **收尾阶段**: 数据清理、存档

### 8.2 资源分配
- **人力资源**: 研究团队、收集员
- **时间资源**: 各阶段时间分配
- **财务资源**: 设备、激励、交通
- **设备资源**: 硬件、软件、材料

## 9. 质量保证措施

### 9.1 信度措施
- **重测信度**: 时间稳定性
- **内部一致性**: 项目间一致性
- **评分者信度**: 评价者间一致性

### 9.2 效度措施
- **内容效度**: 测量内容的代表性
- **结构效度**: 测量结构的准确性
- **准则效度**: 与其他标准的相关性

## 10. 常见错误与避免
- **工具不当**: 選免使用不适合的工具
- **样本偏差**: 選免样本选择偏差
- **数据质量问题**: 選免数据收集质量问题
- **伦理违规**: 選免伦理问题

## 11. 专业术语
- **混杂变量**: 同时影响自变量和因变量的变量
- **选择偏倚**: 样本选择过程中的系统性偏差
- **信息偏倚**: 数据收集过程中的系统性偏差
- **回忆偏倚**: 受试者记忆不准确导致的偏差
- **社会期望偏倚**: 受试者迎合社会期望的倾向
