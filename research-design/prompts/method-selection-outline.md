# 方法选择详纲

## 1. 研究目的导向

### 1.1 探索性研究
- **目的**: 了解未知现象，产生假设
- **适用方法**: 定性研究、案例研究、现象学研究
- **数据类型**: 定性数据为主
- **样本特征**: 小样本，深入分析
- **分析方式**: 主题分析、内容分析、扎根理论

### 1.2 描述性研究
- **目的**: 描述现象的特征和分布
- **适用方法**: 调查研究、横断面研究、观察研究
- **数据类型**: 定量数据为主
- **样本特征**: 大样本，代表性强
- **分析方式**: 描述性统计、频率分析

### 1.3 解释性研究
- **目的**: 解释变量间的关系和因果机制
- **适用方法**: 实验研究、准实验研究、相关研究
- **数据类型**: 定量数据为主
- **样本特征**: 随机抽样，控制变量
- **分析方式**: 推断性统计、回归分析、因果推断

### 1.4 评价性研究
- **目的**: 评估干预措施的效果
- **适用方法**: 实验研究、准实验研究、前后对比
- **数据类型**: 定量数据为主
- **样本特征**: 实验组和对照组
- **分析方式**: t检验、方差分析、效果量计算

## 2. 研究设计类型

### 2.1 定性研究设计
- **现象学研究**: 探索个体对特定现象的体验
  - 适用场景: 理解主观体验和意义
  - 数据收集: 深度访谈、反思日记
  - 分析方法: 现象学分析、主题分析
- **扎根理论**: 从数据中构建理论
  - 适用场景: 理论缺失或需要重构理论
  - 数据收集: 开放式访谈、观察
  - 分析方法: 开放式编码、轴心编码、选择性编码
- **案例研究**: 深入分析特定案例
  - 适用场景: 复杂现象在自然情境中的表现
  - 数据收集: 多源数据（访谈、观察、文档）
  - 分析方法: 案例内分析、跨案例分析
- **民族志**: 研究文化和社会群体
  - 适用场景: 文化和社会行为研究
  - 数据收集: 参与观察、田野笔记
  - 分析方法: 文化分析、象征意义分析
- **叙事研究**: 通过故事理解经验
  - 适用场景: 个人经历和生命历程研究
  - 数据收集: 生命故事、自传
  - 分析方法: 叙事分析、情节分析

### 2.2 定量研究设计
- **实验设计**: 随机对照试验、随机对照试验
  - 特征: 随机分配、控制组、操纵自变量
  - 优点: 因果推断能力强
  - 局限: 外部效度可能受限
- **准实验设计**: 无随机分配的实验
  - 特征: 无随机分配，但有对照组
  - 优点: 现实情境中可行
  - 局限: 内部效度可能受影响
- **相关设计**: 探索变量间关系
  - 特征: 观察变量间关系，不操纵变量
  - 优点: 描述变量间关系
  - 局限: 无法确定因果关系
- **调查设计**: 横断面、纵向调查
  - 特征: 收集大样本数据
  - 优点: 结果推广性强
  - 局限: 因果推断能力弱

### 2.3 混合方法设计
- **收敛式并行设计**: 同时收集和分析定量与定性数据
  - 特征: 两种数据类型同等重要
  - 适用: 验证和补充研究发现
- **解释性顺序设计**: 先定量后定性
  - 特征: 定量结果引导定性探索
  - 适用: 解释定量结果
- **探索性顺序设计**: 先定性后定量
  - 特征: 定性发现指导定量验证
  - 适用: 构建量表或理论模型
- **嵌入式设计**: 一种方法嵌入另一种方法中
  - 特征: 主次方法明确
  - 适用: 辅助主方法的验证

## 3. 选择标准

### 3.1 研究问题匹配
- **问题类型**: 方法是否适合回答研究问题？
- **数据需求**: 数据类型是否匹配研究目标？
- **理论框架**: 方法是否与理论框架一致？
- **假设检验**: 是否能有效检验理论假设？

### 3.2 资源约束
- **时间资源**: 时间是否充足完成研究？
- **财务资源**: 预算是否足够支持研究？
- **人力资源**: 人员是否具备相应技能？
- **技术资源**: 设备和工具是否具备？

### 3.3 伦理考量
- **知情同意**: 研究是否符合知情同意原则？
- **隐私保护**: 是否保护参与者隐私？
- **风险评估**: 研究风险是否在可接受范围内？
- **伦理审批**: 是否通过伦理委员会审批？

## 4. 数据收集方法

### 4.1 定量数据收集
- **问卷调查**: 大样本、标准化测量
  - 优点: 效率高、标准化
  - 局限: 深度有限
- **实验测量**: 控制环境下的精确测量
  - 优点: 精确、控制
  - 局限: 生态效度
- **观察记录**: 结构化观察和计数
  - 优点: 客观、实时
  - 局限: 可能存在观察者偏倚
- **档案数据**: 现有数据库和记录
  - 优点: 成本低、历史数据
  - 局限: 可能不完全符合研究需求

### 4.2 定性数据收集
- **深度访谈**: 个体详细观点和体验
  - 优点: 深入、详细
  - 局限: 样本量小、耗时
- **焦点小组**: 群体动态和互动
  - 优点: 互动、动态
  - 局限: 可能存在群体压力
- **参与观察**: 自然环境中的行为观察
  - 优点: 自然、真实
  - 局限: 观察者偏倚、伦理问题
- **文件分析**: 文档、图像、视频等
  - 优点: 历史、客观
  - 局限: 解释主观性

## 5. 样本策略

### 5.1 概率抽样
- **简单随机抽样**: 每个个体等概率被选中
- **分层抽样**: 按特征分层后抽样
- **整群抽样**: 以群体为单位抽样
- **系统抽样**: 按固定间隔抽样

### 5.2 非概率抽样
- **方便抽样**: 易于接触的个体
- **目的抽样**: 根据特定标准选择
- **雪球抽样**: 通过现有参与者寻找新参与者
- **理论抽样**: 为理论构建选择案例

## 6. 信效度考虑

### 6.1 信度
- **重测信度**: 时间稳定性
- **内部一致性**: 项目间一致性
- **评分者信度**: 评价者间一致性

### 6.2 效度
- **内容效度**: 测量内容的代表性
- **结构效度**: 测量结构的准确性
- **准则效度**: 与其他标准的相关性

## 7. 常见错误与避免
- **设计不当**: 選免研究设计与问题不匹配
- **样本不足**: 選免样本量不足的问题
- **控制不当**: 選免对无关变量控制不当
- **伦理疏忽**: 選免忽视伦理问题

## 8. 专业术语
- **内部效度**: 研究结果归因于自变量的程度
- **外部效度**: 研究结果推广到其他情境的程度
- **建构效度**: 理论建构与测量的一致性程度
- **统计结论效度**: 统计推断的准确性程度
- **三角验证**: 使用多种方法验证结果
