# 中国社会网络中心性分析指南

> **渐进式学习**：本文档提供中国社会网络的特殊性和中心性解释。

## 中国社会网络的特点

### 1. 关系（Guanxi）网络

**定义**：基于人情、面子和互惠的社会关系网络

**特点**：
- 强调长期互惠
- 重视面子和人情
- 关系强度差异大（强关系 vs 弱关系）

**对中心性的影响**：
- 度中心性：反映关系网络的广度
- 介数中心性：反映"中间人"角色
- 特征向量中心性：反映与权威人物的关系

---

### 2. 差序格局

**定义**：以自我为中心的同心圆式社会结构（费孝通）

**特点**：
- 关系亲疏有别
- 圈子文化明显
- 内外有别

**对中心性的影响**：
- 需要区分不同圈子的中心性
- 桥梁节点连接不同圈子
- 核心圈的中心性更重要

---

### 3. 集体主义

**定义**：强调群体利益和归属

**特点**：
- 重视群体和谐
- 强调集体决策
- 领导权威性强

**对中心性的影响**：
- 特征向量中心性更能反映实际影响力
- 度中心性可能低估权威人物的影响
- 需要考虑正式权力和非正式影响力

---

## 中心性在中国社会网络中的解释

### 度中心性（关系广度）

**中国语境**：
- 高值："人脉广"、"朋友多"
- 低值："孤立"、"不合群"

**案例**：
```
节点A：度中心性 = 0.80
解释：A认识网络中80%的人，是典型的"社交达人"，
      在中国文化中被视为"人脉广"、"吃得开"。
```

---

### 介数中心性（中间人角色）

**中国语境**：
- 高值："中间人"、"牵线搭桥"
- 低值："局外人"

**案例**：
```
节点B：介数中心性 = 0.65
解释：B连接不同的关系圈子，扮演"中间人"角色，
      在中国社会中这种角色尤为重要，因为"关系"
      的建立往往需要中间人引荐。
```

---

### 特征向量中心性（影响力）

**中国语境**：
- 高值："有背景"、"靠山硬"
- 低值："没关系"

**案例**：
```
节点C：特征向量中心性 = 0.90
解释：C虽然认识的人不多（度中心性0.30），
      但都是重要人物。在中国文化中，这种
      "质量高于数量"的关系网络更有价值。
```

---

## 本土化分析建议

### 1. 区分强弱关系

使用边权重区分：
- 强关系（weight ≥ 0.7）：家人、密友
- 弱关系（weight < 0.7）：泛泛之交

**工具支持**：
```bash
python scripts/calculate_centrality.py \
  --input network.json \
  --weighted  # 考虑边权重
```

---

### 2. 识别关键角色

**中国社会网络的典型角色**：

| 角色 | 中心性特征 | 中文描述 |
|-----|-----------|---------|
| 社交达人 | 度中心性高 | "人脉广" |
| 中间人 | 介数中心性高 | "牵线搭桥" |
| 意见领袖 | 特征向量中心性高 | "有影响力" |
| 核心人物 | 多种中心性都高 | "关键人物" |

---

### 3. 考虑文化因素

**面子文化**：
- 高中心性节点更注重维护面子
- 失去中心性可能导致"丢面子"

**人情文化**：
- 中心性高的节点承担更多人情债
- 需要平衡给予和索取

**权威文化**：
- 正式权力可能与中心性不一致
- 需要同时分析正式和非正式网络

## 实践案例

**研究**：某企业员工关系网络

**发现**：
- 部门经理（节点A）：度中心性0.45，特征向量中心性0.85
- 解释：虽然A认识的人不多，但都是其他部门的关键人物，
        体现了中国企业中"关系质量"重于"关系数量"
