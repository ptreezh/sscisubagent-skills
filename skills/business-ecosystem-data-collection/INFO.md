# 商业生态系统数据搜集技能 (Business Ecosystem Data Collection Skill)

## 概述
商业生态系统数据搜集技能是一个专门用于搜集商业生态系统分析所需数据的AI技能。该技能能够从真实外部数据源搜集企业信息、关系数据、行业信息等，为后续的生态系统分析提供高质量数据支持。

## 核心功能
该技能具有以下核心功能：
- 从真实外部数据源搜索和识别目标行业内的关键实体（企业、机构等）
- 搜集实体的详细信息（业务模式、规模、市场地位等）
- 识别和映射实体间的关系（供应关系、投资关系、合作关系等）
- 搜集行业趋势、市场规模、竞争格局等宏观信息
- 验证数据质量和完整性

## 详细功能描述

### 1. 多源真实数据搜集
- **百度搜索**：通过爬虫技术从百度搜索获取企业基本信息
- **企业官网**：从企业官网提取详细业务信息
- **新闻资讯**：从新闻网站获取企业合作、投资等关系信息
- **政府公开信息**：从政府网站获取企业注册、资质等信息
- **行业报告**：从免费行业报告中提取市场数据

### 2. 数据搜集范围
- **地理范围**：支持全球、国家、地区等不同地理范围的数据搜集
- **实体类型**：涵盖企业、监管机构、投资机构、供应商、客户等各类实体
- **数据类型**：包括公司概况、财务数据、商业模式、关系网络、市场定位等

### 3. 搜集深度
- **基本深度**：搜集实体基本信息和主要关系
- **标准深度**：搜集详细业务信息和完整的关联关系
- **全面深度**：搜集深度市场分析、竞争格局、行业趋势等信息

### 4. 数据验证
- 实体信息完整性验证
- 关系数据准确性验证  
- 行业信息时效性验证
- 多源数据交叉验证
- 整体数据质量评估

## 使用场景
1. **商业生态系统分析**：为生态系统结构分析提供基础数据
2. **行业竞争分析**：了解行业内主要参与者及其关系
3. **市场进入策略**：识别潜在合作伙伴和竞争对手
4. **投资决策支持**：评估行业格局和投资机会

## 输入参数
- `targetIndustry`：目标行业
- `searchScope`：搜索范围（地理、时间、实体类型）
- `dataTypes`：需要搜集的数据类型
- `collectionDepth`：搜集深度
- `verificationRequired`：是否需要数据验证

## 输出结果
- 搜集到的实体列表
- 实体间关系网络
- 行业宏观信息
- 数据质量报告
- 改进建议

- 数据质量验证和完整性检查
- 支持多种真实数据源和搜集策略
- 具备数据去重和验证机制
