# 学术文献检索技能

## 🎯 功能定位

**重要声明**: 本技能**不提供任何文献下载功能**，仅提供合法合规的文献检索指导和数据库使用策略。

## 📚 核心能力

### 1. 检索策略构建
- **关键词分析**: 提炼核心概念和相关术语
- **检索式设计**: 构建布尔逻辑检索表达式
- **数据库选择**: 推荐最适合的学术数据库
- **检索优化**: 提高检索效率和准确性

### 2. 合法访问指导
- **机构资源**: 指导利用学校/机构订阅的数据库
- **开放获取**: 推荐免费的开放获取学术资源
- **图书馆服务**: 介绍图书馆文献传递服务
- **学术社交网络**: ResearchGate、Academia.edu等平台使用

### 3. 中文数据库指导
- **知网使用**: 正确使用CNKI数据库的方法
- **万方策略**: 万方数据库检索技巧
- **维普期刊**: 维普数据库使用指导
- **读秀学术**: 读秀图书资源利用

### 4. 替代方案推荐
- **开放获取期刊**: Directory of Open Access Journals (DOAJ)
- **预印本服务器**: arXiv、bioRxiv、medRxiv等
- **机构仓储**: 各大学机构的学术成果库
- **政府资源**: 政府报告、统计数据等公开资源

## 🔧 实用工具集成

### 检索工具
- **文献管理软件**: Zotero、EndNote、Mendeley
- **浏览器插件**: Zotero Connector、Google Scholar按钮
- **检索辅助**: CNKI E-Study、NoteExpress
- **引用管理**: CSL格式支持

### 数据库访问方式
```bash
# 机构访问 (推荐)
校园网/机构IP → 直接访问订阅数据库

# VPN访问
学校VPN → 远程访问图书馆资源

# 图书馆卡
公共图书馆 → 部分数据库访问权限

# 开放获取
DOAJ、arXiv → 免费学术资源
```

## 📋 检索流程指导

### 第1步: 需求分析
```
用户输入研究主题 → 分析核心概念
↓
识别相关关键词和同义词
↓
确定学科领域和数据库范围
```

### 第2步: 数据库选择
```
中文学术文献:
- 优先推荐: 知网、万方、维普 (需要机构访问)
- 备选方案: 国家哲学社会科学文献库、中国社会科学网

英文学术文献:
- 综合数据库: Web of Science、Scopus、Google Scholar
- 专业数据库: ERIC、PsycINFO、Sociological Abstracts
- 开放获取: DOAJ、arXiv、bioRxiv
```

### 第3步: 检索式构建
```
基础检索式:
(关键词1 AND 关键词2) OR (同义词1 AND 同义词2)

高级检索式:
((概念A OR 概念A') AND (概念B OR 概念B')) NOT 排除词

字段限定:
标题:(关键词) AND 摘要:(关键词) AND 年限:(2020-2024)
```

### 第4步: 结果优化
```
检索结果过多:
- 增加关键词限制
- 缩短时间范围
- 添加字段限定

检索结果过少:
- 减少关键词限制
- 扩展同义词范围
- 延长时间范围
- 尝试不同数据库
```

## 🏫 机构资源利用指导

### 高校图书馆资源
```bash
# 学生/教职工权限
校园网IP → 免费访问所有订阅数据库
图书馆网站 → 电子资源导航
VPN服务 → 校外访问权限

# 使用步骤
1. 登录学校图书馆网站
2. 找到"数据库导航"或"电子资源"
3. 选择目标数据库
4. 使用学号/工号登录
```

### 公共图书馆资源
```bash
# 市民权限
图书馆卡 → 部分数据库访问权限
现场使用 → 图书馆内免费访问
数字图书馆 → 线上资源服务
```

- **DOAJ**: 开放获取期刊目录

- **bioRxiv**: 生物学预印本
- **medRxiv**: 医学预印本

- **中国社会科学网**: 部分免费内容
- **中国科学院文献情报中心**: 开放获取资源
- **中国科技论文在线**: 免费科技论文

## 📖 使用示例

### 示例1: 大学生文献检索
```
用户: "我是教育学研究生，需要找关于在线学习的文献"

指导:
1. 推荐数据库: CNKI、万方、ERIC、Web of Science
2. 检索式: ("在线学习" OR "网络教育") AND ("效果" OR "影响")
3. 开放获取: DOAJ、ERIC免费资源
4. 机构资源: 学校图书馆VPN访问
```

### 示例2: 国外文献获取
```
用户: "需要查找国外的社会网络分析研究"

指导:
1. 英文数据库: Web of Science、Scopus
2. 关键词: "social network analysis" AND ("China" OR "Chinese")
3. 开放获取: arXiv、SSRN
4. 文献传递: 通过图书馆文献传递服务
```
