# 数字涂尔干社会学分析技能

## 核心功能
- 社会事实特征识别与分析
- 集体意识表征与功能研究
- 社会结构功能分析
- 社会团结类型判断与变迁分析

## 分析流程

### 步骤1: 社会事实识别
**目标**: 识别研究对象的社会事实特征
**操作**:
1. 分析社会事实的外在性特征
2. 评估强制性影响程度
3. 验证独立性属性
4. 分类社会事实类型

**输出格式**:
```
社会事实: [事实名称]
外在性: [高/中/低]
强制性: [强/中/弱]
独立性: [完全/部分/有限]
类型: [制度/规范/价值/行为]
```

### 步骤2: 集体意识分析
**目标**: 分析相关的集体意识表现形式
**操作**:
1. 识别集体表征形式
2. 分析共享价值体系
3. 研究社会规范形成机制
4. 监测集体情感变化

### 步骤3: 功能分析
**目标**: 分析社会现象的功能和结构位置
**操作**:
1. 识别显功能与潜功能
2. 分析功能替代可能性
3. 评估功能整合程度
4. 诊断功能失调问题

### 步骤4: 社会团结判断
**目标**: 判断社会团结类型和变迁机制
**操作**:
1. 分析社会分化程度
2. 评估有机团结特征
3. 识别机械团结要素
4. 预测社会变迁趋势

## 标准化输出

### 必需输出
1. **社会事实分析报告**: 完整的社会事实特征分析
2. **集体意识研究报告**: 集体表征和价值体系分析
3. **功能分析报告**: 功能结构和功能关系分析
4. **社会团结评估**: 团结类型和变迁趋势评估

### 可选输出
1. **比较分析报告**: 不同社会形态的功能差异
2. **历史分析报告**: 社会事实演变过程追踪
3. **统计分析报告**: 社会现象相关性量化分析
4. **案例研究报告**: 典型社会事实深入分析
