# 范畴构建完整案例

> **渐进式披露**：本文档提供范畴构建的完整案例。理论基础见`paradigm-theory.md`，故障排除见`troubleshooting.md`。

## 研究背景

**研究主题**：大学新生的学业适应过程

**数据来源**：
- 10名大一新生的深度访谈（每人60-90分钟）
- 3次焦点小组讨论
- 学生的学习日志（2个月）

**开放编码结果**：识别出32个初始概念

### 人工精炼范畴

**精炼原则**：
1. 检查概念归类的合理性
2. 完善范畴命名（行动导向、理论意义）
3. 撰写清晰的范畴定义
4. 提供典型案例支持

**精炼后的范畴1**：
```
范畴名称：主动寻求学业支持
范畴定义：学生在遇到学业困难时，主动向教师、同学、学长等寻求帮助和指导的行为模式。

包含的概念（8个）：
1. 寻求教师指导 - 向任课教师请教问题
2. 参加答疑 - 利用教师答疑时间
3. 请教学长 - 向高年级学生请教经验
4. 加入学习小组 - 参与同学组织的学习小组
5. 寻找学习伙伴 - 寻找志同道合的学习伙伴
6. 利用学业辅导中心 - 使用学校提供的辅导服务
7. 在线求助 - 通过网络平台寻求帮助
8. 请教家人 - 向家人咨询学习问题

典型案例：
"我刚开始学高数的时候完全听不懂，课后我就去找老师答疑。老师很耐心地给我讲解，还推荐我去学业辅导中心。后来我还加入了一个学习小组，大家互相帮助，我的成绩慢慢提高了。"（学生A，访谈第3次）

属性维度（见analyze_properties.py输出）：
- 求助对象：教师/同学/学长/家人
- 求助方式：面对面/在线/小组
- 求助频率：高频/中频/低频
- 求助主动性：主动/被动
```

**精炼后的范畴2**：
```
范畴名称：调整学习策略
范畴定义：学生根据大学学习特点，主动调整学习方法、时间管理和学习习惯的过程。

包含的概念（7个）：
1. 制定学习计划 - 制定周计划、日计划
2. 调整学习方法 - 从题海战术转向理解为主
3. 时间管理 - 使用时间管理工具和技巧
4. 自我监控 - 定期检查学习进度和效果
5. 课前预习 - 养成预习习惯
6. 课后复习 - 及时复习巩固
7. 笔记整理 - 整理和完善课堂笔记

典型案例：
"高中时我就是刷题，但大学不行。我开始尝试理解概念，而不是死记硬背。我还制定了每周的学习计划，用番茄工作法来管理时间。虽然一开始很不习惯，但坚持下来后效果很好。"（学生B，访谈第5次）

属性维度：
- 调整领域：方法/时间/习惯
- 调整程度：大幅/中度/微调
- 调整效果：显著/一般/不明显
- 调整持续性：持续/间断/放弃
```

**精炼后的范畴3**：
```
范畴名称：建立社会支持网络
范畴定义：学生通过建立友谊、加入社团、维护宿舍关系等方式，构建情感和学业支持网络的过程。

包含的概念（6个）：
1. 结交新朋友 - 主动认识新同学
2. 建立学习伙伴 - 找到学习上的互助伙伴
3. 加入社团 - 参加学生社团活动
4. 维护宿舍关系 - 与室友建立良好关系
5. 保持家庭联系 - 与家人保持沟通
6. 寻求情感支持 - 向朋友倾诉压力和困扰

典型案例：
"刚来的时候很孤独，后来我加入了一个社团，认识了很多朋友。我们不仅一起玩，还经常一起学习。有了这些朋友，我感觉适应快多了。宿舍关系也很重要，室友之间互相帮助，有问题可以一起讨论。"（学生C，访谈第4次）

属性维度：
- 网络类型：学业/情感/娱乐
- 网络规模：大/中/小
- 网络质量：深厚/一般/浅层
- 网络稳定性：稳定/波动/不稳定
```

**精炼后的范畴4**：
```
范畴名称：认知和情绪调整
范畴定义：学生通过调整对自己和学业的认知评价，以及管理负面情绪，来适应大学学习的心理过程。

包含的概念（5个）：
1. 接受不完美 - 接受自己不是"最优秀"的
2. 降低期望 - 调整对成绩的不合理期望
3. 积极归因 - 将失败归因为可控因素
4. 情绪调节 - 管理焦虑、压力等负面情绪
5. 自我肯定 - 肯定自己的进步和努力

典型案例：
"高中时我一直是班里前几名，来到这里发现高手如云。一开始我很沮丧，觉得自己不行。后来我慢慢接受了这个现实，不再要求自己必须第一。我开始关注自己的进步，而不是和别人比较。心态调整后，反而学得更好了。"（学生D，访谈第6次）

属性维度：
- 调整对象：自我认知/期望/归因/情绪
- 调整方向：积极/消极
- 调整效果：有效/无效
- 调整稳定性：稳定/反复
```

**精炼后的范畴5**：
```
范畴名称：学业适应困难体验
范畴定义：学生在适应大学学习过程中体验到的各种负面情绪和心理困扰。

包含的概念（6个）：
1. 感到压力 - 学业压力、竞争压力
2. 学习焦虑 - 对考试、作业的焦虑
3. 自我怀疑 - 怀疑自己的能力
4. 孤独感 - 缺乏社会支持的孤独
5. 迷茫感 - 对未来和专业的迷茫
6. 挫败感 - 努力后仍失败的挫败

典型案例：
"第一学期真的很难熬。我每天都很焦虑，担心考试不及格。看到别人都学得很好，我就怀疑自己是不是不适合这个专业。有时候晚上一个人在宿舍，就会感到很孤独，很想家。"（学生E，访谈第2次）

属性维度：
- 情绪类型：焦虑/压力/孤独/挫败
- 情绪强度：强/中/弱
- 持续时间：长期/间歇/短暂
- 应对方式：主动/回避/求助
```

## 第二步：建立范畴层级

### 识别核心范畴和次要范畴

**核心范畴**（proportion > 15%）：
- C1：主动寻求学业支持（25%）
- C2：调整学习策略（22%）
- C5：学业适应困难体验（19%）

**次要范畴**（proportion ≤ 15%）：
- C3：建立社会支持网络（19%，接近核心）
- C4：认知和情绪调整（16%）

**层级关系**：
```
核心现象：学业适应困难体验（C5）
├── 应对策略（核心）
│   ├── 主动寻求学业支持（C1）
│   └── 调整学习策略（C2）
└── 支持性策略（次要）
    ├── 建立社会支持网络（C3）
    └── 认知和情绪调整（C4）
```

### 撰写范畴关系备忘录

**备忘录示例**：
```
日期：2025-12-18
主题：核心现象的确定

经过反复比较，我认为"学业适应困难体验"（C5）应该作为核心现象，理由如下：

1. 数据支持：所有10名受访者都提到了适应困难，这是最普遍的体验。

2. 理论意义：适应困难是触发其他所有行动的关键。如果没有困难，学生就不会采取应对策略。

3. 逻辑关系：其他4个范畴都是对适应困难的回应：
   - C1和C2是直接的应对策略
   - C3和C4是支持性策略
   
4. 时间序列：访谈显示，学生首先体验到困难（C5），然后才采取行动（C1-C4）。

但也有疑问：C5是一个"体验"而不是"行动"，这是否符合Strauss的Paradigm模型？需要进一步思考。

可能的替代方案：将C1"主动寻求学业支持"作为核心现象？但这样就无法解释为什么学生会寻求支持。

决定：暂定C5为核心现象，继续分析。
```

## 第三步：范畴命名的优化

### 命名原则

**1. 行动导向**：
- ❌ 不好："学业支持"（名词）
- ✅ 好："主动寻求学业支持"（动词短语）

**2. 理论意义**：
- ❌ 不好："学习方法调整"（过于具体）
- ✅ 好："调整学习策略"（有理论抽象性）

**3. 中文语境**：
- ❌ 不好："Seeking Academic Support"（英文直译）
- ✅ 好："主动寻求学业支持"（符合中文表达习惯）

### 命名对比表

| 初始命名（工具建议） | 优化后命名 | 优化理由 |
|------------------|----------|---------|
| 寻求学业支持 | 主动寻求学业支持 | 强调"主动性"，突出行动特征 |
| 调整学习方法 | 调整学习策略 | "策略"比"方法"更有理论意义 |
| 建立社会支持 | 建立社会支持网络 | "网络"强调关系的系统性 |
| 心态调整 | 认知和情绪调整 | 更精确地描述调整的内容 |
| 适应困难体验 | 学业适应困难体验 | 明确"学业"领域，避免过于宽泛 |

## 第四步：验证范畴质量

### 使用工具进行质量检查

```bash
# 假设有一个validate_categories.py脚本
python scripts/validate_categories.py --input categories_refined.json --output validation.json
```

**验证维度**：
1. **内部一致性**：范畴内的概念是否相关？
2. **外部区分度**：范畴间是否有明确区分？
3. **数据支持**：每个范畴是否有充分的数据支持？
4. **理论意义**：范畴是否有理论价值？

### 人工验证示例

**验证范畴1：主动寻求学业支持**

✅ **内部一致性检查**：
- 所有8个概念都涉及"寻求帮助"
- 都是主动行为（不是被动接受）
- 都与学业相关（不是生活或情感）

✅ **外部区分度检查**：
- 与C2（调整学习策略）的区别：C1是寻求外部帮助，C2是自我调整
- 与C3（建立社会支持网络）的区别：C1是学业支持，C3是情感支持

✅ **数据支持检查**：
- 10名受访者中有9人提到寻求帮助
- 共有45个相关文本片段
- 覆盖了访谈的不同阶段

✅ **理论意义检查**：
- 体现了"主动适应"的理论概念
- 与社会支持理论相关
- 有助于解释适应的个体差异

**结论**：范畴1质量良好，保留。

## 关键经验总结

### 成功经验

1. **工具与人工结合**：工具提供初步聚类，人工进行理论精炼
2. **持续比较**：不断在范畴间、范畴内进行比较
3. **备忘录记录**：及时记录思考过程和理论洞察
4. **数据回溯**：反复回到原始数据验证范畴

### 常见陷阱

1. **过度依赖工具**：工具只是辅助，不能替代理论判断
2. **范畴过多**：初学者容易创建过多范畴，要敢于合并
3. **命名不当**：使用过于具体或过于抽象的名称
4. **忽视数据**：范畴构建要有充分的数据支持
