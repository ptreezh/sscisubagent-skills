# 商业模式画布分析技能提示词

## 主要提示词
请对以下企业进行商业模式画布分析，基于权威的商业模式画布理论框架分析九个核心要素。确保分析结果准确、全面且基于真实可靠的信息源。

### 分析要求
1. **关键合作伙伴**：识别企业的重要合作伙伴和供应商
2. **关键活动**：分析企业核心业务活动
3. **关键资源**：识别企业核心资源和资产
4. **价值主张**：分析企业为客户创造的价值
5. **客户关系**：分析企业与客户的关系类型
6. **渠道通路**：分析企业的产品/服务交付渠道
7. **客户细分**：识别企业的目标客户群体
8. **成本结构**：分析企业成本构成
9. **收入来源**：分析企业收入模式

### 数据验证要求
- 从权威数据源收集信息
- 多源数据交叉验证
- 确保信息时效性和准确性
- 标注所有信息来源

### 输出格式
```json
{
  "keyPartners": ["合作伙伴1", "合作伙伴2"],
  "keyActivities": ["关键活动1", "关键活动2"], 
  "keyResources": ["核心资源1", "核心资源2"],
  "valuePropositions": ["价值主张1", "价值主张2"],
  "customerRelationships": ["客户关系1", "客户关系2"],
  "channels": ["渠道1", "渠道2"],
  "customerSegments": ["客户细分1", "客户细分2"],
  "costStructure": ["成本构成1", "成本构成2"],
  "revenueStreams": ["收入来源1", "收入来源2"],
  "dataSources": {
    "officialWebsite": "企业官网",
    "annualReport": "年度报告",
    "governmentDatabase": "政府数据库"
  }
}
```

### 质量控制
- 检查九个要素是否完整分析
- 验证各要素间的逻辑一致性
- 确保分析结果与企业实际情况相符
- 评估信息源的可信度

## 子技能提示词

### 1. 数据收集提示词
从权威渠道收集企业相关信息，包括：
- 官方网站信息
- 年度报告数据
- 行业研究报告
- 新闻媒体资料

验证数据源的权威性和可信度，确保信息准确性。

### 2. 元素分析提示词
针对每个商业模式画布元素，进行深入分析：
- 分析其在整体商业模式中的作用
- 评估其重要性和影响力
- 识别相关的机会和挑战

### 3. 关联性分析提示词
分析九个要素之间的相互关系：
- 价值主张如何通过渠道传递给客户
- 关键资源如何支持关键活动
- 收入来源如何覆盖成本结构
