# 商业模式画布分析技能

## 概述
该技能基于权威的商业模式画布理论（Osterwalder & Pigneur, 2010），对企业进行全面的商业模式分析。分析内容包括商业模式画布的九个核心要素。

## 功能特性
- **九要素分析**：完整分析商业模式画布的九个要素
- **权威理论基础**：基于经过验证的商业模式理论
- **数据驱动**：从权威数据源收集信息进行分析
- **验证机制**：多重验证确保分析结果准确性

## 输入要求
- `companyData`：企业基础信息对象，包含：
  - `name`: 企业名称
  - `industry`: 所属行业
  - `products/services`: 产品/服务信息
  - `revenueModel`: 收入模式（可选）
  - `marketPosition`: 市场地位（可选）

## 输出结果
分析结果包含以下九个要素：
1. **关键合作伙伴** (Key Partners)
2. **关键活动** (Key Activities) 
3. **关键资源** (Key Resources)
4. **价值主张** (Value Propositions)
5. **客户关系** (Customer Relationships)
6. **渠道通路** (Channels)
7. **客户细分** (Customer Segments)
8. **成本结构** (Cost Structure)
9. **收入来源** (Revenue Streams)

## 使用场景
- 企业战略规划
- 商业模式评估
- 竞争分析
- 投资决策支持
- 管理咨询

## 理论基础
基于《商业模式新生代》(Business Model Generation) 一书中的商业模式画布框架，该框架由亚历山大·奥斯特瓦德(Alexander Osterwalder)和伊夫·皮尼厄(Yves Pigneur)开发。

## 数据源
- 企业官方网站
- 年度报告
- 政府数据库
- 金融数据服务商
- 行业研究报告

## 准确性保证
- 多源数据交叉验证
- 权威性评分机制
- 逻辑一致性检查
- 专家知识库验证
