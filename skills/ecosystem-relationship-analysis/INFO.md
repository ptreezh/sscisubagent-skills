# 生态系统关系分析技能 (Ecosystem Relationship Analysis Skill)

## 核心信息
- **技能ID**: ecosystem-relationship-analysis
- **版本**: 2.0.0
- **类型**: 生态系统分析
- **用途**: 分析商业生态系统中各主体间的关系，包括合作、竞争、供应、投资、监管等多种关系类型

## 概述
该技能专门用于分析商业生态系统中各实体间的关系网络，提供关系类型分析、网络拓扑分析、关键关系识别、生态系统健康度评估等功能。

## 核心功能
1. **关系类型分析**: 分析合作、竞争、供应、投资、监管等多类关系
2. **网络拓扑分析**: 计算网络密度、中心性、连通性等拓扑属性
3. **关键关系识别**: 识别对生态系统影响最大的关键关系
4. **健康度评估**: 评估生态系统的整体健康状况
5. **社区检测**: 识别生态系统中的子群落或模块化结构

## 输入输出规范
- **输入**: entities(实体列表), relationships(关系列表), analysisDepth(分析深度)
- **输出**: 关系分析结果、网络属性、生态系统健康度、优化建议

## 使用场景
- 商业生态系统建模
- 供应链网络分析
- 合作伙伴关系评估
- 竞争格局分析
- 生态系统稳定性评估
