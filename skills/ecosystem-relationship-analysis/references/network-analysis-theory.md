# 网络分析理论

## 网络拓扑指标

### 基础指标
- **度（Degree）**: 节点连接数，反映节点的直接连接数
- **度中心性（Degree Centrality）**: 节点在网络中的直接影响力
- **接近中心性（Closeness Centrality）**: 节点到其他节点的平均距离
- **中介中心性（Betweenness Centrality）**: 节点在最短路径中的中介作用

### 高级指标
- **聚类系数（Clustering Coefficient）**: 衡量节点邻居间的连接密度
- **网络密度（Network Density）**: 实际连接数与可能连接数的比例
- **平均路径长度（Average Path Length）**: 网络中任意两点间的平均距离

## 网络模型
- **随机网络模型**: 用于对比真实网络的基准模型
- **小世界网络**: 具有高聚类系数和短路径长度的网络
- **无标度网络**: 度分布遵循幂律分布的网络
