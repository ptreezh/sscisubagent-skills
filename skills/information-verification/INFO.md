# 信息验证专家技能 - AI优化参考文档

## 技能核心信息
- **技能ID**: information-verification
- **版本**: 2.0.0
- **分类**: 信息验证
- **类型**: 验证与评估类技能

## 技能概述（简明扼要）
信息验证专家技能是商业模型系统分析智能体的重要组成部分，专门负责对企业信息来源、准确性、可信度和时效性进行全面验证。该技能基于信息科学和验证理论，采用多种验证方法和技术，确保分析所依据的信息真实、准确、可靠。

## 核心功能（要点式）
1. 来源验证 - 验证信息来源的权威性和可靠性
2. 准确性检查 - 验证信息内容的准确性
3. 可信度评估 - 评估信息的可信度和可靠性
4. 数据完整性 - 检查数据是否完整无缺
5. 时效性验证 - 验证信息的时效性和更新状态
6. 综合验证 - 执行全面的信息验证流程

## 输入输出规范
### 输入参数
- `information`: 待验证的信息对象（必需）
- `verificationType`: 验证类型（必需）
- `config`: 验证配置选项（可选）

### 输出结果
- `verification`: 验证对象及结果
- `recommendations`: 验证改进建议
- `dataQualityReport`: 数据质量报告

## 执行流程（步骤化）
1. **输入验证** - 验证输入参数的完整性和有效性
2. **类型判断** - 根据验证类型选择相应模块
3. **多维验证** - 执行来源、准确性、完整性等验证
4. **结果聚合** - 综合各项验证结果
5. **报告生成** - 根据深度要求生成不同详细程度报告

## 关键算法与方法
- **来源可信度评估** - 基于权威性、声誉、独立性、透明度
- **准确性验证算法** - 逻辑一致性检查、格式验证
- **可信度评估模型** - 一致性、合理性、可验证性、文档性
- **渐进式报告生成** - 根据深度参数生成不同详细程度报告

## 应用场景
- 商业模型分析前的数据验证
- 企业信息质量评估
- 多源数据交叉验证
- 决策支持系统数据保障

## 常见使用模式
- 基础验证：快速验证信息基本质量
- 标准验证：全面验证信息质量
- 深度验证：多维度深入验证
- 综合验证：执行所有验证维度

## 关键配置选项
- `depth`: 验证深度（basic/standard/comprehensive）
- `authoritativeSources`: 权威信息源列表
- `toleranceLevel`: 容错水平（0-1）

## 评估标准
- **置信度阈值**: 70%以上为验证通过
- **风险等级**: 基于置信度和问题严重性评估
- **完整性要求**: 关键字段必须存在

## 与其他技能的交互
- 可被其他分析类技能调用进行数据预处理
- 与数据收集类技能配合使用
- 为决策类技能提供数据质量保障
