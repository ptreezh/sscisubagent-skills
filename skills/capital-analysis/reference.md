# 资本分析参考文档

## 1. 金融理论基础

### 1.1 资本结构理论
- **MM理论**（Modigliani-Miller Theorem）
  - 无税MM理论：资本结构与公司价值无关
  - 有税MM理论：债务税盾效应增加公司价值
  - 现实中的权衡理论：权衡税盾收益与破产成本
- **代理理论**
  - 债务的代理收益：减少自由现金流的代理成本
  - 债务的代理成本：债权人与股东的利益冲突
- **信息不对称理论**
  - 优序融资理论：内源融资优先，债务次之，股权最后

### 1.2 投资理论
- **现代投资组合理论**（马科维茨）
  - 风险与收益的权衡
  - 分散化降低非系统风险
- **资本资产定价模型**（CAPM）
  - 系统性风险与预期收益的关系
  - 资本成本的计算
- **套利定价理论**（APT）
  - 多因子模型解释资产收益

### 1.3 公司金融理论
- **股利政策理论**
  - 股利无关论（MM理论）
  - 股利相关论（税差理论、信号理论、代理理论）
- **企业价值评估理论**
  - 现金流折现模型
  - 相对估值法
  - 实物期权理论

## 2. 财务分析框架

### 2.1 资本结构分析
#### 关键比率
- **负债比率** = 总负债 / 总资产
- **权益乘数** = 总资产 / 股东权益
- **负债权益比** = 总负债 / 股东权益
- **长期负债比率** = 长期负债 / 总资产

#### 分析维度
- 行业对比分析
- 历史趋势分析  
- 绝对水平分析

### 2.2 盈利能力分析
#### 盈利能力比率
- **资产收益率**（ROA）= 净利润 / 总资产
- **净资产收益率**（ROE）= 净利润 / 股东权益
- **销售净利率** = 净利润 / 营业收入
- **资产周转率** = 营业收入 / 总资产

### 2.3 偿付能力分析
#### 短期偿债能力
- **流动比率** = 流动资产 / 流动负债
- **速动比率** = (流动资产 - 存货) / 流动负债
- **现金比率** = 现金及现金等价物 / 流动负债

#### 长期偿债能力
- **利息保障倍数** = 息税前利润 / 利息费用
- **负债权益比** = 总负债 / 股东权益

## 3. 估值方法

### 3.1 现金流折现法
- **自由现金流模型**（FCFF & FCFE）
- **股利折现模型**（DDM）
- **经济增加值模型**（EVA）

### 3.2 相对估值法
- **市盈率**（P/E）= 股价 / 每股收益
- **市净率**（P/B）= 股价 / 每股净资产
- **企业价值倍数**（EV/EBITDA）
- **市销率**（P/S）= 股价 / 每股销售收入

### 3.3 资产基础法
- **清算价值法**
- **重置成本法**
- **净资产价值法**

## 4. 风险管理

### 4.1 财务风险识别
- **信用风险**：交易对手违约风险
- **市场风险**：利率、汇率、商品价格波动
- **流动性风险**：无法及时变现的风险
- **操作风险**：内部流程和系统风险

### 4.2 风险度量方法
- **VaR**（风险价值）：在特定置信水平下的最大可能损失
- **压力测试**：极端情况下的风险暴露
- **敏感性分析**：关键变量变化对价值的影响

## 5. 融资策略

### 5.1 融资方式选择
- **股权融资**：IPO、增发、私募股权
- **债务融资**：银行贷款、公司债券、可转债
- **混合融资**：优先股、可转债

### 5.2 融资时机选择
- **市场时机理论**：利用市场错误定价
- **融资窗口**：选择合适的市场条件
- **生命周期匹配**：融资方式与企业发展阶段匹配

## 6. 投资决策

### 6.1 投资项目评估
- **净现值**（NPV）：NPV > 0 项目可行
- **内部收益率**（IRR）：IRR > 要求回报率
- **投资回收期**：收回初始投资的时间

### 6.2 资本配置
- **战略投资**：与核心业务相关的投资
- **财务投资**：追求财务回报的投资
- **风险投资**：高风险高回报的投资
