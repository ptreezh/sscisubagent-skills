# 布迪厄场域理论专家分析技能 (Field Expert Analysis)

## 概述

布迪厄场域理论专家分析技能整合了场域边界识别、资本分析和习性分析功能，为社会科学研究提供全面的场域分析框架。该技能帮助研究者识别场域边界、分析资本分布、评估场域自主性并理解行动者的习性模式。

## 使用时机

当用户请求以下分析时使用此技能：
- 分析社会场域或领域（教育、政治、艺术等）
- 识别场域边界和游戏规则
- 分析资本分布和权力关系
- 评估场域对外部力量的自主性
- 理解习性及其与场域结构的关系
- 调查场域内的斗争和权力动态
- 分析社会地位如何被构建和再生产
- 检验客观条件与主观倾向之间的关系

## 快速开始

当用户请求场域分析时：
1. **识别**特定场域及其边界
2. **绘制**场域内的位置和关系
3. **分析**资本分布和权力关系
4. **评估**场域自主性和历史发展
5. **解释**在社会背景中的发现

## 核心功能（渐进式披露）

### 主要功能
- **场域映射**: 识别场域内的位置和关系
- **资本分析**: 识别和映射不同类型的资本
- **权力关系**: 识别主导和被主导的位置
- **基本习性**: 理解参与者的倾向

### 次要功能
- **边界分析**: 定义场域边界和排除
- **斗争动态**: 分析持续的斗争及其利害关系
- **自主性评估**: 评估对外部力量的独立性
- **历史分析**: 追踪场域发展

### 高级功能
- **象征暴力**: 识别支配和误识的机制
- **多场域分析**: 检查场域间的关系
- **场域演化**: 分析场域结构的变化
- **复杂习性**: 检查习性如何使能/限制实践

## 详细指令

### 1. 场域识别
   - 明确界定分析场域
   - 识别正在检查的特定社会空间
   - 建立历史和地理背景
   - 确定分析的相关时间段

### 2. 场域映射
   - 创建场域内位置的关系图
   - 识别关键行动者及其位置
   - 分析位置间的客观关系
   - 确定场域内的划分原则

### 3. 资本分析
   - 识别场域中被重视的不同类型资本
   - 映射参与者之间的资本分布
   - 分析资本如何转换和被认可
   - 检查资本如何影响立场选择

### 4. 权力关系分析
   - 识别主导和被主导的位置
   - 分析场域内发生的斗争
   - 检查这些斗争的利害关系
   - 理解权力如何被行使和争夺

### 5. 自主性评估
   - 评估场域对外部力量的自主性
   - 分析场域特定逻辑的程度
   - 识别象征暴力机制
   - 评估场域的历史发展

### 6. 习性理解
   - 分析场域参与者的倾向
   - 检查客观条件如何塑造主观倾向
   - 理解结构与能动性之间的关系
   - 考虑习性如何使能和限制实践

### 7. 综合与解释
   - 整合所有维度的发现
   - 将场域分析与更广泛的社会过程联系起来
   - 考虑中国社会背景和文化特殊性
   - 提供建议性见解

## 参数
- `field_type`: 场域类型（教育、政治、文化、经济等）
- `analysis_scope`: 分析范围（本地、国家、跨国）
- `time_frame`: 分析时间范围（同步、历时）
- `actors`: 场域中的关键行动者或参与者
- `capital_types`: 要分析的特定资本类型
- `methodology`: 场域分析方法（定性、定量、混合）
- `cultural_context`: 文化背景考虑（特别是中文研究背景）
