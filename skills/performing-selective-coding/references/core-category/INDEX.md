# 核心范畴识别指南

> **渐进式学习**：本文档提供核心范畴识别的概览和方法。

## 什么是核心范畴

核心范畴（Core Category）是扎根理论中最重要的范畴，能够统领和整合所有其他范畴，形成理论的核心。

**Glaser (1978) 的定义**：
> "核心范畴是研究的中心主题，所有其他范畴都与它相关，它能够解释大部分的变异。"

---

## 选择标准（Strauss & Corbin, 1998）

### 1. 解释力（Explanatory Power）
- 能够解释大部分研究现象
- 覆盖≥60%的数据

### 2. 关联度（Connectivity）
- 与多个其他范畴有密切关系
- 参与≥20%的范畴关系

### 3. 数据支持（Data Support）
- 有丰富的实证数据支持
- 数据片段数≥50个

### 4. 理论价值（Theoretical Significance）
- 具有重要的理论贡献
- 能够推进学术对话

---

## 识别方法

### 方法1：量化评估（使用工具）

```bash
python scripts/identify_core_category.py \
  --input categories.json \
  --relations relationships.json \
  --top 3
```

**工具输出**：
```json
{
  "summary": {
    "core_category_names": ["学业适应困难", "主动寻求支持", "调整学习策略"],
    "top_score": 0.78
  }
}
```

### 方法2：网络中心性分析

使用NetworkX计算范畴的中心性：
- 度中心性：连接数量
- 中介中心性：桥接作用
- 特征向量中心性：影响力

### 方法3：定性判断

询问关键问题：
- "这个研究最核心的问题是什么？"
- "哪个范畴能够统领其他范畴？"
- "如果只能选一个范畴，会是哪个？"

## 典型案例

**研究**：大学生学业适应

**候选核心范畴**：
1. 学业适应困难（现象）- 评分0.78
2. 主动寻求学业支持（策略）- 评分0.72
3. 调整学习策略（策略）- 评分0.68

**最终选择**：学业适应困难

**理由**：
- ✅ 解释力强：所有10名受访者都提到
- ✅ 关联度高：与其他4个范畴都相关
- ✅ 数据支持：45个数据片段
- ✅ 理论价值：适应理论的核心概念

## 验证核心范畴

### 检查清单

- [ ] 能够统领其他范畴
- [ ] 与研究问题契合
- [ ] 有充分的数据支持
- [ ] 具有理论意义
- [ ] 能够形成连贯的故事线

### 使用工具验证

```bash
python scripts/identify_core_category.py \
  --input categories.json \
  --relations relationships.json \
  --criteria criteria.json  # 自定义验证标准
```

## 常见问题

**问题1**：多个范畴评分相近，难以选择
- 解决：综合考虑研究问题和理论贡献

**问题2**：核心范畴过于抽象
- 解决：选择更具体、更有操作性的范畴

**问题3**：核心范畴与研究问题不匹配
- 解决：重新审视研究问题，或调整核心范畴
