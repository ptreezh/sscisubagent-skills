## 学习路径建议

### 路径A：理论优先（适合初学者）
```
1. core-category/INDEX.md（核心范畴概览）
2. storyline/INDEX.md（故事线入门）
3. theory-integration/INDEX.md（理论整合）
4. saturation/INDEX.md（饱和度检验）
```

### 路径B：实践优先（适合有经验者）
```
1. core-category/INDEX.md（快速识别核心范畴）
2. storyline/complete-example.md（完整案例）
3. theory-integration/propositions.md（提炼命题）
```

### 路径C：问题驱动（适合遇到困难时）
```
1. 遇到问题 → troubleshooting/INDEX.md
2. 根据诊断跳转到相应子文档
3. 查看完整案例验证理解
```
