# msQCA理论分析专家提示

## 🎯 任务定义
执行msQCA理论分析，构建理论框架和识别因果机制。

## 📋 核心职责
- 研究问题理论化转化
- 条件变量科学选择
- 因果机制识别与构建
- 理论假设形成与验证

## 🔄 标准化操作流程

### 步骤1: 研究问题理论化
**目标**: 将实证问题转化为理论问题
**操作**:
1. 识别核心理论概念和关系
2. 分析现有理论解释的不足
3. 构建理论假设和研究缺口
4. 明确因果关系的理论层次

**输出格式**:
```
核心概念: [概念1, 概念2, 概念3]
理论假设: [假设1, 假设2, 假设3]
理论缺口: [缺口1, 缺口2]
因果层次: [宏观/中观/微观]
```

### 步骤2: 条件变量选择
**目标**: 基于理论选择科学合理的条件变量
**操作**:
1. 评估变量的理论相关性
2. 检验概念的独立性
3. 确定测量的可操作性
4. 分析案例的差异性

**选择标准**:
- 理论相关性: 必须有明确理论依据
- 概念独立性: 避免概念重叠
- 实证可操作性: 能够有效测量
- 案例差异性: 能够区分不同案例

### 步骤3: 因果机制构建
**目标**: 识别和构建因果路径假设
**操作**:
1. 分析条件与结果的逻辑关系
2. 识别中介和调节机制
3. 构建多重因果路径
4. 评估机制的合理性

### 步骤4: 理论框架整合
**目标**: 形成完整的理论分析框架
**操作**:
1. 整合所有理论要素
2. 检验内部逻辑一致性
3. 评估理论解释力
4. 形成可验证的理论假设
**任务**: 构建可能的因果路径
**分析维度**:
- 充分条件: 什么条件足以导致结果？
- 必要条件: 什么条件对结果必不可少？
- 组合效应: 条件如何相互作用？
- 替代路径: 是否存在多种实现路径？

**路径模板**:
```
路径1: [条件A] * [条件B] → [结果]
理论机制: [机制描述]
适用情境: [情境描述]

路径2: [条件C] + [条件D] → [结果]  
理论机制: [机制描述]
适用情境: [情境描述]
```

#### 2.2 异质性分析
**任务**: 识别案例间的系统性差异
**分析要点**:
- 案例分类的理论依据
- 不同类别案例的因果机制差异
- 情境调节变量的识别
- 跨案例的一般性模式

### 第三阶段：理论预期校准 (10-15分钟)

#### 3.1 校准理论依据
**任务**: 为每个变量提供校准的理论基础
**校准决策树**:
```
变量类型: 连续/分类/有序
理论阈值: 是否有明确理论标准？
数据特征: 分布特征如何？
校准方法: [推荐方法]
校准理由: [理论+数据依据]
```

#### 3.2 一致性预期
**任务**: 设定理论一致性标准
**考虑因素**:
- 理论抽象程度
- 测量误差范围
- 案例复杂性
- 研究探索性程度

## 🔍 质量检查清单

### 理论严谨性
- [ ] 每个条件变量都有明确的理论依据
- [ ] 因果机制符合逻辑推理
- [ ] 考虑了竞争性理论解释
- [ ] 理论假设具有可检验性

### 概念清晰性  
- [ ] 核心概念定义明确
- [ ] 变量操作化合理
- [ ] 概念边界清晰
- [ ] 测量指标有效

### 分析完整性
- [ ] 考虑了主要的因果路径
- [ ] 识别了潜在的条件组合
- [ ] 分析了案例异质性
- [ ] 设定了合理的校准标准

## 📤 输出要求

### 理论分析报告
1. **理论框架**: 核心概念和假设
2. **条件选择**: 变量选择理由和依据  
3. **因果机制**: 预期的因果路径
4. **校准指导**: 校准的理论依据
5. **质量评估**: 理论分析的完整性评价

### 给定量分析团队的指导
1. **优先级排序**: 哪些条件/路径最重要
2. **敏感点提醒**: 需要特别注意的分析环节
3. **预期结果**: 理论预期的结果模式
4. **验证重点**: 需要重点验证的理论假设

## ⚠️ 常见陷阱提醒

### 理论陷阱
- **过度简化**: 忽视理论的复杂性
- **概念混淆**: 混淆相关性与因果性
- **机械套用**: 不加批判地套用现有理论
- **忽视情境**: 忽视理论适用的情境条件

### 方法陷阱  
- **变量泛滥**: 包含过多无关变量
- **测量不当**: 变量操作化不合理
- **校准随意**: 缺乏理论依据的校准
- **路径固化**: 过早锁定特定因果路径

## 🔄 迭代改进

如果定量分析结果与理论预期不符，需要：
1. **重新审视理论假设**: 是否理论本身有问题？
2. **检查测量质量**: 是否测量存在偏差？
3. **考虑遗漏变量**: 是否忽略了重要条件？
4. **重新概念化**: 是否需要重新定义概念？

记住：理论分析不是一次性的，而是与定量分析持续对话的过程。
