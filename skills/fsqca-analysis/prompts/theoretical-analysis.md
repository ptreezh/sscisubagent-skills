# fsQCA理论分析

## 任务目标
构建fsQCA研究的理论框架，识别因果机制，为后续校准和定量分析奠定基础。

## 核心任务
1. **研究问题澄清** - 明确研究目标和变量
2. **理论框架构建** - 确定条件变量和因果假设
3. **概念操作化** - 将概念转化为可测量指标
4. **校准策略规划** - 为每个条件确定校准方法

## 简化分析流程
1. **问题澄清** → 研究问题、结果变量(Y)、前因条件(X1..Xn)
2. **框架构建** → 核心/边界/控制条件，因果路径假设
3. **操作化** → 概念定义、理论含义、极值状态
4. **校准规划** → 方法选择（直接/间接/阈值等）

## 输出要求
- 条件选择的理论依据
- 预期因果组态类型
- 每个变量的校准方法推荐

## 质量检查
- [ ] 理论框架逻辑一致？
- [ ] 条件选择有理论依据？
- [ ] 校准策略可行？

## 关键术语
- **组态**: 条件的特定组合
- **模糊集**: 允许不同程度隶属的集合
- **隶属度**: 0到1之间的隶属程度
