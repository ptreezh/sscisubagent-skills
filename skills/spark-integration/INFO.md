# 科大讯飞星火大模型集成技能 - AI优化参考文档

## 技能核心信息
- **技能ID**: spark-integration
- **版本**: 1.0.0
- **分类**: AI集成
- **类型**: 大模型集成与调用类技能

## 技能概述（简明扼要）
科大讯飞星火大模型集成技能提供与科大讯飞星火大模型的API集成能力，支持调用讯飞星火系列模型进行智能分析和对话。该技能封装了API调用细节，为其他技能提供统一的AI能力接口。

## 核心功能（要点式）
1. API调用封装 - 封装科大讯飞API调用细节
2. 模型选择 - 支持多种讯飞星火模型
3. 智能分析 - 提供文本分析和理解能力
4. 对话功能 - 支持智能对话交互
5. 参数管理 - 管理API密钥和模型参数

## 输入输出规范
### 输入参数
- `prompt`: 输入的提示词（必需）
- `model`: 要使用的模型（可选，默认spark-max）

### 输出结果
- `response`: API响应内容
- `success`: 执行状态
- `metadata`: 调用元数据

## 执行流程（步骤化）
1. **参数验证** - 验证API密钥和输入参数
2. **请求构建** - 构建API请求参数
3. **API调用** - 调用科大讯飞星火API
4. **响应处理** - 解析和处理API响应
5. **结果返回** - 返回处理后的结果

## 关键算法与方法
- **API请求构建** - 标准化API请求格式
- **错误处理** - 处理API调用异常和错误
- **响应解析** - 解析API响应内容

## 应用场景
- 为其他技能提供AI分析能力
- 智能对话系统集成
- 文本分析和理解任务
- 自然语言处理任务

## 常见使用模式
- 同步调用：直接获取API响应
- 异步处理：处理长时间运行的请求
- 批量处理：批量处理多个请求

## 关键配置选项
- `model`: 指定使用的讯飞模型
- `api_key`: 科大讯飞API密钥

## 评估标准
- **API调用成功**: 成功调用讯飞API
- **响应质量**: 获取高质量AI响应
- **错误处理**: 适当处理API错误

## 与其他技能的交互
- 被其他AI分析技能调用
- 提供底层AI能力支持
- 与认证系统配合管理API密钥
