## 🛠️ 智能化部署方式

### 1. 智能部署器 (推荐) ⭐
```bash
# 一键智能部署
python smart_deploy.py --deploy

# 系统诊断
python smart_deploy.py --diagnose

# 生成使用指南
python smart_deploy.py --guide

# 创建快速启动脚本
python smart_deploy.py --quick-start
```

**优势**:
- 🔍 自动检测系统环境
- 📦 智能选择最佳包管理器 (uv优先)
- ⚙️ 自动初始化jieba配置
- 🛠️ 修复常见依赖问题
- 📊 提供详细的诊断报告

### 2. 命令行启动器
```bash
# 交互式技能选择
python skills_launcher.py

# 快速启动最近使用的技能
python skills_launcher.py --quick

# 显示欢迎界面
python skills_launcher.py --welcome
```

**优势**:
- 🎯 友好的交互式菜单
- 📋 智能技能分类展示
- 🔄 使用历史记录
- 🔍 技能搜索功能
- 📊 批量处理模式

### 3. Web界面 (最直观) 🌐
```bash
# 安装Web依赖 (首次使用)
pip install flask flask-cors

# 启动Web界面
python web_interface.py

# 访问浏览器
# http://127.0.0.1:5000
```

**优势**:
- 🌐 浏览器友好的图形界面
- 📁 拖拽上传数据文件
- 📊 实时显示分析结果
- 💾 一键下载分析报告
- 📱 响应式设计，支持移动端

## 🔧 核心技能使用指南

### 1. 中文开放编码 📝
**适用场景**: 访谈文本分析、概念提取、编码优化

#### Web界面使用:
1. 访问 `/skill/open-coding`
2. 上传访谈文本文件 (.txt)
3. 选择脚本: `preprocess.py` (预处理)
4. 查看结果并下载

#### 命令行使用:
```bash
# 文本预处理
python skills/coding/open-coding/scripts/preprocess.py --input interview.txt

# 概念提取
python skills/coding/open-coding/scripts/extract_concepts.py --input preprocessed.json

# 编码比较
python skills/coding/open-coding/scripts/compare_codes.py --input codes.json
```

### 2. 网络中心性分析 📊
**适用场景**: 社交网络分析、关系研究、影响力评估

#### Web界面使用:
1. 访问 `/skill/centrality-analysis`
2. 上传网络数据文件 (.json)
3. 自动计算4种中心性指标
4. 生成分析报告

#### 命令行使用:
```bash
python skills/analysis/centrality-analysis/scripts/centrality.py --input network.json
```

### 3. 理论饱和度检验 🎯
**适用场景**: 扎根理论验证、编码完整性检查

#### Web界面使用:
1. 访问 `/skill/theory-saturation`
2. 上传编码数据目录
3. 自动评估饱和状态
4. 生成详细报告

#### 命令行使用:
```bash
python skills/coding/theory-saturation/scripts/assess_saturation.py --data-dir data/
```

## 📚 数据准备指南

### 1. 开放编码数据格式
```json
{
  "interview_text": "访谈文本内容",
  "segments": [
    {"id": 1, "text": "第一段内容"},
    {"id": 2, "text": "第二段内容"}
  ]
}
```

### 2. 网络分析数据格式
```json
{
  "nodes": ["节点A", "节点B", "节点C"],
  "edges": [
    ["节点A", "节点B", 1],
    ["节点B", "节点C", 2]
  ]
}
```

### 3. 饱和度检验数据格式
```json
{
  "existing_codes": [
    {"code": "概念A", "frequency": 10},
    {"code": "概念B", "frequency": 8}
  ],
  "new_data": [
    {"text": "新的访谈内容", "concepts": ["新概念"]},
    {"text": "更多内容", "concepts": ["其他概念"]}
  ]
}
```
