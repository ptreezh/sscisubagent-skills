## 📚 文献检索逻辑

### 1️⃣ 检索策略构建逻辑

#### 步骤1: 需求分析
```
用户输入研究主题
    ↓
提取核心概念
    ↓
识别相关关键词和同义词
    ↓
确定学科领域
    ↓
设定时间范围
    ↓
明确文献类型要求
```

**示例**:
```
输入: "数字鸿沟的社会学影响"
分析:
- 核心概念: 数字鸿沟、社会学
- 关键词: 数字鸿沟、数字不平等、信息不平等
- 学科: 社会学
- 相关词: 社会分层、社会公平、信息社会
```

#### 步骤2: 数据库选择策略

**中文数据库推荐顺序**:
1. **知网(CNKI)** - 最全面的中文数据库
2. **万方数据** - 期刊、会议、学位论文
3. **维普期刊** - 中文科技期刊
4. **国家哲学社会科学文献库** - 免费开放

**英文数据库推荐顺序**:
1. **Web of Science** - 高质量引文数据库
2. **Scopus** - 综合性摘要数据库
3. **Google Scholar** - 免费搜索引擎
4. **学科专业数据库** - ERIC、PsycINFO等

#### 步骤3: 检索式构建

**基础逻辑**:
```boolean
(关键词1 AND 关键词2) OR (同义词1 AND 同义词2)
```

**高级逻辑**:
```boolean
((概念A OR 概念A同义词) AND (概念B OR 概念B同义词))
NOT 排除词
AND 年限:(2020-2024)
AND 文献类型:(期刊论文)
```

**字段限定**:
```
标题:(关键词)  - 在标题中搜索
摘要:(关键词)  - 在摘要中搜索
作者:(姓名)    - 按作者搜索
机构:(单位)    - 按机构搜索
基金:(项目号)  - 按基金资助搜索
```

#### 步骤4: 结果优化

**结果过多的处理**:
```python
# 增加限制
- 添加更多关键词
- 缩短时间范围
- 限定在标题字段搜索
- 增加排除词
```

**结果过少的处理**:
```python
# 扩展范围
- 减少关键词限制
- 扩展同义词范围
- 延长时间范围
- 改用全文搜索
- 尝试其他数据库
```

## 🏫 支持的检索方式

### 方式1: 机构资源访问（推荐）

#### 适用对象
- 高校学生、教职工
- 研究机构研究人员
- 拥有机构订阅的人员

#### 访问方式
```bash
# 校园网内
1. 连接校园网/教育网
2. 访问图书馆数据库导航
3. 选择目标数据库
4. 直接访问（IP认证）

# 校园网外
1. 使用学校VPN
2. 登录图书馆系统
3. 通过VPN访问数据库
4. 使用机构账号登录
```

#### 优势
- ✅ 完全免费
- ✅ 可下载全文
- ✅ 访问所有订阅资源
- ✅ 合法合规

---

### 方式2: 公共图书馆资源

#### 适用对象
- 普通市民
- 没有机构订阅的独立研究者

#### 访问方式
```bash
# 办理图书馆卡
1. 到当地公共图书馆办理读者卡
2. 激活数字资源访问权限
3. 通过图书馆网站访问数据库
4. 部分数据库可现场使用
```

#### 可访问资源
- 中国知网（部分）
- 万方数据（部分）
- 维普（部分）
- 龙源期刊
- 超星电子书

---

### 方式3: 开放获取资源（Open Access）

#### 国际开放获取数据库

**预印本服务器**:
- **arXiv.org** - 物理、数学、计算机科学
- **bioRxiv** - 生物学
- **medRxiv** - 医学
- **SSRN** - 社会科学

**开放获取期刊目录**:
- **DOAJ** (Directory of Open Access Journals)
- **PLOS ONE** - 多学科开放获取
- **Scientific Reports** - Nature出版集团

#### 中文开放获取资源

**国家级平台**:
- **国家哲学社会科学文献库** - 免费注册
  - 网址: https://www.nssd.cn/
  - 特点: 完全免费，注册后可下载

- **中国社会科学网**
  - 网址: http://www.cssn.cn/
  - 特点: 部分免费内容

- **中国科学院文献情报中心**
  - 网址: http://www.las.ac.cn/
  - 特点: 开放获取资源

**机构开放仓储**:
- 各大学机构知识库
- 中国科技论文在线
- 中国预印本服务系统

---

### 方式4: 文献传递服务

#### 服务流程
```bash
1. 通过学校图书馆申请
2. 提供文献详细信息
3. 图书馆馆际互借
4. 获取文献（通常1-3天）
5. 可能产生一定费用
```

#### 适用场景
- 机构没有订阅的特定期刊
- 需要获取稀有文献
- 研究急需特定文献

## 📖 具体使用指导

### 场景1: 大学生文献检索

**需求**: "我是教育学研究生，需要找关于在线学习的文献"

**指导步骤**:

1. **数据库选择**
   ```
   中文: 知网、万方、维普
   英文: ERIC、Web of Science
   免费: 国家哲学社会科学文献库
   ```

2. **检索式构建**
   ```
   ("在线学习" OR "网络教育" OR "远程教育")
   AND
   ("效果" OR "影响" OR "成效")
   AND
   ("高等教育" OR "大学")
   ```

3. **限定条件**
   ```
   - 时间: 2019-2024
   - 文献类型: 期刊论文
   - 期刊等级: CSSCI、北大核心
   ```

4. **获取方式**
   ```
   - 通过学校VPN访问知网
   - 或使用国家哲学社会科学文献库
   ```

---

### 场景2: 独立研究者文献获取

**需求**: "我没有学校账号，如何获取社会学文献？"

**推荐方案**:

1. **免费注册国家哲学社会科学文献库**
   ```
   网址: https://www.nssd.cn/
   步骤: 注册 → 登录 → 搜索 → 下载
   特点: 完全免费，文献质量高
   ```

2. **使用公共图书馆资源**
   ```
   - 办理市图书馆读者卡
   - 访问图书馆数字资源
   - 现场使用数据库
   ```

3. **开放获取期刊**
   ```
   - DOAJ搜索开放获取期刊
   - arXiv预印本（理工科）
   - SSRN（社会科学）
   ```

4. **直接联系作者**
   ```
   - ResearchGate请求全文
   - Academia.edu获取
   - 邮件联系作者索取
   ```

## 💡 最佳实践建议

### 检索策略
1. **从宽泛到精确**: 先用少量关键词检索，再逐步增加限制
2. **多数据库对比**: 不同数据库覆盖范围不同
3. **保存检索历史**: 记录有效的检索式
4. **定期更新**: 关注最新研究进展

### 文献管理
1. **使用文献管理软件**: Zotero、EndNote等
2. **建立分类体系**: 按主题、项目等分类
3. **及时添加笔记**: 记录阅读心得和引用要点
4. **定期备份**: 防止数据丢失

### 合法使用
1. **尊重版权**: 不大规模传播文献
2. **合理引用**: 遵循引用规范
3. **个人使用**: 下载的文献仅供个人研究
4. **支持作者**: 条件允许时购买正版
